<p>So the news is out!  </p>
<p><a href="http://silverlight.net"><font size="3"><strong>Silverlight</strong></font></a><font size="3"><strong> IS the platform for Windows Phone 7 Series development! </strong></font></p>
<p>Sweet.  We also made available an <a href="http://timheuer.com/blog/archive/2010/03/15/whats-new-in-silverlight-4-rc-mix10.aspx">update to Silverlight 4</a> that you might be interested in too!</p>
<p><img align="left" src="http://storage.timheuer.com/wp7series-225.png" alt="Windows Phone 7 Series" title="Windows Phone 7 Series" style="margin: 0px 20px; display: inline;" />You may be wondering how you get started.  If you are new to Silverlight, I recommend getting familiar with Silverlight first.  You can find all the tools you will need at the <a href="http://silverlight.net/getstarted">Silverlight community site</a>.  In addition to the core tools you’ll want to get the Windows Phone Developer Tools CTP.  This will add to your Visual Studio 2010 installation (or install Visual Studio Express) to enable Windows Phone and XNA Game Studio development.  Be sure to read the documentation on the release notes to understand any limitations.  A link to the tools, documentation, developer/UX guides and more can be found on the <strong><a href="http://silverlight.net/getstarted/devices/windows-phone/">Silverlight for Windows Phone</a></strong> page.  The key elements you’d want to get  are:</p>
<ul>
    <li><a href="http://go.microsoft.com/fwlink/?LinkID=185584">Windows Phone developer tools CTP</a> (read the <a href="http://go.microsoft.com/fwlink/?LinkID=185269">release notes</a>) </li>
    <li><a href="http://electricbeach.org/?p=438">Expression Blend 4 add-on for Windows Phone</a> (Christian Schormann’s blog) </li>
</ul>
<p>I’ve also taken some quick time to get some quick videos up for some tips and familiarity with the tools and some initial areas you’ll want to take a look at.  Here are some starting videos for you:</p>
<ul>
    <li><a href="http://silverlight.net/learn/videos/all/get-started-with-silverlight-for-windows-phone">Getting started with Silverlight and Windows Phone development</a> </li>
    <li><a href="http://silverlight.net/learn/videos/all/windows-phone-application-bar">Windows Phone Application Bar</a> </li>
    <li><a href="http://silverlight.net/learn/videos/all/windows-phone-silverlight-navigation">Navigation framework for Windows Phone applications</a> </li>
    <li><a href="http://silverlight.net/learn/videos/all/windows-phone-splash-screen">Custom application splash screen</a></li>
    <li><a href="http://channel9.msdn.com/learn/courses/WP7TrainingKit/">Windows Phone 7 Series Developer Training Kit</a></li>
</ul>
<p>The getting started video has some quick tips and tricks about the emulator and using the keyboard input control (referred to as the ‘SIP’).  I suggest taking a look at these for some primer.  If you have questions afterwards, check out the <a href="http://forums.silverlight.net/forums/63.aspx">dedicated forum for Silverlight for Windows Phone</a>.</p>
<p>I’m looking forward to seeing what you’ll develop using Silverlight for Windows Phone!  Be sure also to watch for <a href="http://twitter.com/ckindel">@ckindel</a> and <a href="http://twitter.com/wp7dev">@wp7dev</a> on <a href="http://twitter.com/timheuer">Twitter</a> for information about Windows Phone 7 Series development.</p>
