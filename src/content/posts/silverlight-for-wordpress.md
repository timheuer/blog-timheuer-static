<p>After looking at the options available for WordPress plugins for <a href="http://silverlight.net/">Silverlight</a>, I found one that was outdated and the author’s URL looked to be broken.  So I adapted/updated the plugin to work again.</p>
<p>I’ve now provided the <strong>Silverlight Plugin for WordPress</strong> for an easy way to host Silverlight applications within your WordPress content.  You can get the plugin via the WordPress plugin directory by searching on Silverlight or <a href="http://wordpress.org/extend/plugins/silverlight-for-wordpress/">clicking here</a>.<br />
</p>
<p>Once downloaded, unzip the contents in your /wp-content/plugins directory.  You’ll then be able to activate the plugin and start working with it.  Adding content is as simple as adding this text to your post:</p>
<blockquote>
<p>[silverlight: myapp.xap]</p>
</blockquote>
<p>And you will have your content in your post.  You can optionally specify the width, height and minimum Silverlight version required.</p>
<p>Version 1.0.3 (31 MAR 2009) includes:</p>
<ul>
    <li>Ability to add local/absolute Silverlight applications (XAP) </li>
    <li>Optionally set width/height/minRuntimeVersion of the Silverlight host plugin </li>
    <li>Optionally provide initParams to the plugin </li>
</ul>
<h2>Silverlight Media Player (new 2010!)</h2>
<p>I also have a <strong>Silverlight Media Player for WordPress</strong> which is built on top of the <a href="http://smf.codeplex.com">Silverlight Media Framework</a>.  It is also as simple as tagging content and supports any Silverlight-supported video type, including IIS Smooth Streaming:</p>
<blockquote>
<p>[sl-media: http://mysite.com/myhomemovie.mp4]</p>
</blockquote>
<p>You can find this and the installation/configuration details page in the WordPress plugin directory as well!</p>
