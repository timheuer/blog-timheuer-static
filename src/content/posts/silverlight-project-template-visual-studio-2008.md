<p>i've got a few messages about 'hey, where did the <strong><a rel="tag" href="http://silverlight.net/">silverlight</a></strong> project template go in visual studio 2008?  what the heck, doesn't microsoft want us to create projects?'</p>
<p>trouble is that it is there, but different...you know same, but different.</p>
<p>if you have <strong>visual studio 2008</strong> installed, make sure you install the <a href="http://www.microsoft.com/downloads/details.aspx?FamilyId=25144C27-6514-4AD4-8BCB-E2E051416E03&amp;displaylang=en">silverlight tools for visual studio 2008</a>.  yeah, i know it says 'tools for alpha' -- just install it would ya.  after you install it, you'll have the project templates for silverlight <strike>1.1</strike> 2.0 as well as for 1.0.  but where is it?</p>
<p>File...New Web Site... then you should see this:</p>
<p><img alt="" src="http://s3.amazonaws.com/timheuer-img/scriptable-web.png" /></p>
<p>that's what you want.  hope this helps.</p>
