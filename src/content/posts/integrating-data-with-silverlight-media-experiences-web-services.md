<p>a video screencast that <a href="http://silverlight.net/blogs/jesseliberty/">jesse</a> and i collaborated on <a href="http://silverlight.net/learn/learnvideo.aspx?video=573">was just posted</a>.  in the screencast jesse demonstrates the code we did to demonstrate a concept he calls "hypervideo" which is basically leveraging media metadata, and simulating retrieving related data information to display in the experience.</p>
<p><a href="http://silverlight.net/learn/learnvideo.aspx?video=573"><img style="MARGIN: 0px 5px 0px 0px" align="left" border="0" alt="" src="http://s3.amazonaws.com/timheuer-img/hypervid-small.png" /></a>this concept is very similar to the experience at <a href="http://hsn.tv">HSN.tv</a> of showing "related products" using the metadata being streamed in their experience.  i hope you find it useful and if you have any questions, please let me know!</p>
<p>again, my goal is to try to help provide you with relevant information about microsoft web platform technologies like <a rel="tag" href="http://silverlight.net">silverlight</a> and enabling scenarios.  if you have a good idea (and aren't exactly sure how to go about it), send it my way as i'd love to help you and the broader community learn how to do it.</p>
<p>are these fully complete samples?  probably not.  they are intended to help you think and work through a portion of potentially your use of these technologies.  if they suck, let me know.</p>
