<p>Last week I had the pleasure of talking with the crew from the <a href="http://herdingcode.com/?p=69">Herding Code podcast</a>.  Among the questions was one about the open source community and <a href="http://silverlight.net">Silverlight</a>.  There was mention about MVCContrib and how that community sprouted almost instantly with the ASP.NET MVC initial releases.  I let the Herding Code crew know that I believe there already is a vibrant open source community around Silverlight and pointed to things like <a href="http://www.devexpress.com/Products/NET/Controls/Silverlight/Grid/">DevExpress’ AgDataGrid</a> control which is free and provides source.  I also mentioned a project that <strong><em>did</em></strong> start right away around Silverlight 2 initial releases: <a href="http://www.codeplex.com/silverlightcontrib" rel="tag">SilverlightContrib</a>.  The project, started by <a href="http://www.pagebrooks.com">Page Brooks</a>, had a good start and keeps on rolling now as they just released a set of <strong>free open source Silverlight controls!</strong></p>  <p><a href="http://silverlightcontrib.org"><img style="display: block; float: none; margin-left: auto; margin-right: auto" border="0" src="http://s3.amazonaws.com:80/timheuer-img/silverlight_contrib.png" /></a></p>  <p>The team of Page Brooks, <a href="http://web-snippets.blogspot.com/">Rob Houweling</a>, <a href="http://firstfloorsoftware.com/">Koen Zwikstra</a>, and <a href="http://www.wildermuth.com">Shawn Wildermuth</a> have just released “Alpha 2” of the project which includes:</p>  <ul>   <ul>     <li>Color Picker</li>      <li>Gauge Control</li>      <li>Star Selector</li>      <li>Enhanced Metafile (EMF)</li>      <li>Cool Menu</li>      <li>Libraries: Zip, Byte utilities, String utilities, Simple text parser, Animation Tweening, Wheel Mouse listener</li>   </ul> </ul>  <p>While the project code is hosted at <a href="http://codeplex.com/silverlightcontrib">Codeplex</a>, the team also has a blog at <a href="http://silverlightcontrib.org">SilverlightContrib.org</a> which has a Live Demo page that will show you demonstrations about the controls:</p>  <p>Color Picker:</p>  <p><img src="http://s3.amazonaws.com:80/timheuer-img/colorpicker.png" /></p>  <p>Cool Menu (OSX style):</p>  <p><img src="http://s3.amazonaws.com:80/timheuer-img/coolmenu.png" /></p>  <p>Gauge samples:</p>  <p><img src="http://s3.amazonaws.com:80/timheuer-img/contirbgauge.png" /></p>  <p>The code is available with the <a href="http://www.opensource.org/licenses/ms-pl.html">Ms-Pl</a> license which is very permissive, so go and grab these controls…and better yet…contribute back to the project!!!  There are more than just controls there as I mentioned there are libraries as well – check out the <a href="http://silverlightcontrib.org/demo/">Live Demo</a> for examples.  I haven’t checked to see their compatibility with the Blend designer, but that is something you can tweak with yourself as needed.</p>  <p>Congratulations to Page and the team for a great community resource and a great release.  I look forward to more great things!</p>  <div style="padding-bottom: 0px; margin: 0px;
