<p>Now that <a href="http://weblogs.asp.net/scottgu/archive/2008/06/06/silverlight-2-beta2-released.aspx">beta 2 is out</a> and some of the features or more solid, and a majority of the breaking changes have been announced, there is no excuse to hold back anymore.  Oh yeah, and there is a <a href="http://go.microsoft.com/fwlink/?Linkid=116610">commercial go-live license available now</a>, so no more excuses :-).</p>
<p>So where to get started?  Well you should first head over to the <a href="http://silverlight.net">Silverlight</a> community site and visit the <a href="http://silverlight.net/GetStarted">Get Started</a> section.  There you will find a rather simple formula to get started:</p>
<ul>
    <ul>
        <li>Download/install the tools </li>
        <li>Watch a quick video (13 minutes) walking through the tools and getting started on your first app </li>
        <li>Read some blog posts </li>
        <li>Watch some videos on different features (<a href="http://silverlight.net/learn">7 new ones for Beta 2</a>) </li>
        <li><strong>Subscribe</strong> to some excellent bloggers:
        <ul>
            <li>Well, I hope <a href="http://feeds.timheuer.com/timheuer">you’ll subscribe here</a>. </li>
            <li><a href="http://silverlight.net/blogs/microsoft/rss.aspx">Silverlight Community Feed</a> </li>
        </ul>
        </li>
    </ul>
</ul>
<p>The community feed includes some great folks like <strong><a href="http://www.irritatedvowel.com">Pete Brown</a></strong>, <strong><a href="http://adoguy.com/">Shawn Wildermuth</a></strong>, <strong><a href="http://geekswithblogs.net/WynApseTechnicalMusings/">Dave Campbell</a></strong> just to name a few.  As an example…all things ADO.NET Data Services (and with Silverlight): Shawn is your man right now.  I was thinking about giving him a hard time about his “ADO Guy” moniker…but he’s holding true to it for now :-).</p>
<p>The <a href="http://silverlight.net/learn">Learn section</a> of the Silverlight community site is intended to help you.  Watch the videos, download the source code, read the tutorials, try the hands-on-labs (thanks <strong><a href="http://blogs.msdn.com/hanuk/">Hanu</a></strong>!), post feedback – and post here what you want to see.  We have a lot in production right now that will be filtering over the next weeks (including some “Silverlight for Business” content) so be sure to stay posted (and subscribed for auto updates).</p>
<p>Once you’ve created that masterpiece, be sure to let us know about it!</p>
