<P>i've just posted the latest version of feedreader to my site for <A href="http://www.timheuer.com/downloads/rssfeedreadersetup.msi">download</A>.&nbsp; there are some things you should know:</P>
<UL>
<LI>there is no compatability between any prior version.&nbsp; yes this sucks, and i'm kinda mad about it myself, but hey, it's free.&nbsp; there are good reasons for this break in compat, so deal with it ;-) 
<LI>new syndication library with broader support for feeds (and improved performance) 
<LI>international language support (yeah, special characters! --vi kanne se v&#229;re omgangsspr&#229;k n&#229;!) 
<LI>removed the branding/copyright from the initial user view (moved to toolpane...hey, it's still free) 
<LI>changed option on how to filter feeds (can use none, days, item count) 
<LI>initial support for Atom 0.3 format -- there may be some kinks with this one, so if you see some, let me know 
<LI>added collapse/expand capabilities for description viewing 
<LI>removed dependency on <A title=SharePoint href="http://www.microsoft.com/sharepoint" target=_blank>sharepoint</A> image files, all self contained
<LI><FONT color=#ff0000>special note: i no longer include a sharepoint installer (just an msi installer to deploy the files)...the reason is simple...you should decide how you want to install it...i won't force you either way.</FONT></LI></UL>
<P>i hope you like it...if you don't, let me know why.</P>
