<p>later tonight, daylight savings time in north america will end, one week later than usual.  you can see some more details on the new DST start/end times on <a href="http://www.microsoft.com/time">http://www.microsoft.com/time</a> (this is a new URL reference for future references to DST/time zone changes/issues beyond 2007).</p>
<p>for a list of microsoft products that might be impacted, see the <a href="http://support.microsoft.com/gp/dst_prodlist">Product Impact Support</a> page as well as the <a href="http://support.microsoft.com/gp/dst_hottopics">DST/TZ hot topics</a> page.  make sure your windows mobile devices have the correct information as well or if older, have had the <a href="http://www.microsoft.com/windowsmobile/daylightsaving/default.mspx">windows mobile patch</a> applied.</p>
<p>for developers, visual studio specific information can be found here:</p>
