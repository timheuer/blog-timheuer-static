<p>thanks to <a href="http://blogs.msdn.com/mikeormond" rel="tag">mike ormond</a> for prodding me about updating the msdn search sidebar gadget.  i had found this issue as well myself when i realized it wasn't working anymore.  well, it was because msdn went through a redesign and their urls changed...so much for scraping :-)</p> <p>well, i updated mine and i'm sure <a href="http://www.nikhilk.net" rel="tag">nikhil</a> is swamped and may not have a chance to update his (and i didn't want to redist his code, but here's what i did).</p> <p>on mine you can just download the new one here: <a href="http://www.box.net/shared/m8xfkrni1r">MSDN Search Sidebar Gadget</a></p> <p>when i say "mine" it is just a basic typical gadget.  <a href="http://www.nikhilk.net/SidebarGadgets.aspx">nikhil's</a> was written using his <a href="http://projects.nikhilk.net/Projects/ScriptSharp.aspx" rel="tag">Script#</a> which is fantastically awesome.  he also has some UI updates.  which one is better?  they both yield the same result...i actually use nikhils because his UI is better...i'm merely updating mine here for the masses (same download link).</p> <p>on to nikhil's changes required.  if you want to just update the currently installed one, go to C:\Users\&lt;youruserid&gt;\AppData\Local\Microsoft\Windows Sidebar\Gadgets and look for the installation point.  on my machine it is called "MSDNSearch.gadget" and open that folder.  in there you will find an MSDNGadget.js file.  go to line 26 and change:</p><pre class="csharpcode">http://search.msdn.microsoft.com/search/data.ashx?&amp;query={0}</pre>
<p>to</p><pre class="csharpcode">http://search.msdn.microsoft.com/search/Feed.aspx?query={0}&amp;feed=rss</pre>
<p>boom, then you are done.  if you want to modify the script# project, then open the MSDNSearch.cs file and make the same change to the SearchUrlFormat private string on line 13.  then you can recompile and all is well.</p>
<p>hope this helps...and thanks for the nudge <a href="http://blogs.msdn.com/mikeormond/archive/2007/06/05/move-over-system-io-packaging-hello-open-xml-sdk.aspx">mike</a>!</p>
