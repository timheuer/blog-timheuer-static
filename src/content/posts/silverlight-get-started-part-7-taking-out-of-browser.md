<p>You can see that after we check for a successful install, we hide the install button as it isn’t needed anymore.  What we need to do though is make that more dynamic in that it automatically knows to do that.  Additionally we don’t want the install button to show if the application is launched out-of-browser to begin with.  Luckily we have an API with a few properties that will help us out: <em>InstallState</em> and <em>IsRunningOutOfBrowser</em>.  Let’s make use of both of these.</p>
