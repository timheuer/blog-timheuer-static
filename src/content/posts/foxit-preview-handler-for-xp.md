<p>Ever since I released my version of the <a target="_blank" href="http://www.timheuer.com/blog/archive/2007/02/27/14001.aspx">Foxit PDF Preview Handler</a>, I've been flooded with comments about building a version that works for Windows XP.  You see XP doesn't have the preview host that Vista has built into the operating system.</p>
<p>Well, I finally had some time to hunt down some code and get it working, thanks to the help of Ryan Gregg from the Outlook team.  I released the sample code I used as a base for this as a part of our <a target="_blank" href="http://thecodetrip.com/1/codegallery-comserver">Code Trip</a> project.  You can view my short screencast explaining one of the core pieces here using the Code Trip's embedded <strong><a target="_blank" rel="tag" href="http://silverlight.net/">Silverlight</a></strong> player:</p>
<p><iframe src="http://thecodetrip.com/1/codegallery-comserver/roadreel" frameborder="0" width="380" scrolling="no" height="310" allowtransparency="allowtransparency"></iframe></p>
<p>You can <a href="http://timheuer.s3.amazonaws.com/FoxPDFPSrv.zip">download the Foxit PDF Preview Handler for XP here</a>.  If you have Windows Vista, I recommend you stick with the existing one for Vista as it leverages the existing preview host that is built into Vista.</p>
<p>Hope this helps!</p>
