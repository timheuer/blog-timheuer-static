&lt;Runtime.InteropServices.ComVisible(<span class="kwrd">True</span>)&gt; _
<span class="kwrd">Public</span> <span class="kwrd">Class</span> RegClass</pre>

<p>the next thing you have to ensure is that the assembly is signed with a <a href="http://msdn2.microsoft.com/en-us/library/wd40t7ad.aspx">strong name</a>.  the reason for this is that when the assembly (dll file) is going to be registered, it will need to have a reference to the codebase or alternatively will be placed in the <a href="http://msdn2.microsoft.com/en-us/library/yf1d93sz.aspx">global assembly cache (GAC)</a> both of which require assemblies to be signed.  once you have those steps complete, you can register the .net assembly with com, using the <a href="http://msdn2.microsoft.com/en-us/library/tzat5yw6(VS.80).aspx">regasm.exe</a> tool that comes with the .net sdk.  to sign the assembly, visual studio makes this easy through the project properties options in the "signing" tab, allowing you to create a new strong name key file at that time as well:</p>
<p><a title="sshot-13" href="http://www.flickr.com/photos/66831573@N00/416892176/"><img alt="sshot-13" src="http://static.flickr.com/172/416892176_b29c520669.jpg" border="0" /></a></p>
<p>once you have this you are ready to use the regasm.exe tool using the command seen here (note the highlighted portion):</p>
<p><a title="sshot-16" href="http://www.flickr.com/photos/66831573@N00/416892212/"><img alt="sshot-16" src="http://static.flickr.com/174/416892212_1ac96d1624.jpg" border="0" /></a></p>
<p>i prefer to use the /codebase option as it allows me to not require it be in the GAC and gives me flexibility of where the assembly may physically reside.</p>
<p>after you have done this you can start calling the com object.  using the sample code (available at the end of this download) you would do something like this in php:</p><pre class="csharpcode">$com_app = <span class="kwrd">new</span> COM(<span class="str">"PHPInteropCSharpClass.SimpleClass"</span>);
$result_var = $com_app-&gt;SayHello(<span class="str">"Woodingo"</span>);
echo $result_var;</pre>
