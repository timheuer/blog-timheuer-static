<p>today, microsoft is announcing a funding of a project to build the <a href="http://sourceforge.net/projects/odf-converter" rel="tag">Open XML Translator</a>, which is a set of tools to translate between office openxml and odf documents. </p>
<p>msft is also announcing an interop center for office 2007, which apparently is an add-in for office 2007 where you'll get PDF XLS and ODF interop. </p>
<p>anyhow, interesting news -- probably moreso that msft is funding this open source effort for three partners for a project hosted on sourceforge...hey, couldn't they at least have said <a href="http://codeplex.com/" rel="tag">codeplex</a>? ;-)</p><span class="tags">
