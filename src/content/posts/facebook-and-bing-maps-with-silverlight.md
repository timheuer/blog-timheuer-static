<p>Two SDK libraries have released this week, helping developers integrate <a href="http://silverlight.net">Silverlight</a> experiences into their applications.</p>  <h2>Bing Maps Silverlight Control</h2>  <p>At TechEd Europe, the Bing Maps team released version 1.0 of the official Bing Maps Silverlight Control!  This is a replacement for the previous Virtual Earth Javascript control and provides a great DeepZoom experience for using the mapping platform.  Here’s some of the great features for Silverlight developers:</p>  <ul>   <li>Built on Silverlight 3! </li>    <li>Designer support for <a href="http://microsoft.com/expression">Expression Blend</a> </li>    <li>Support for out-of-browser </li>    <li>Vector in Pushpin class </li>    <li>Data binding – much needed and welcomed </li>    <li>UI enhancements (animations for simple features) </li>    <li>Embeddable maps </li> </ul>  <p>Great release, and a great addition to any Silverlight developer’s toolbox.</p>  <p align="center"><img title="Bing Maps in Silverlight using Layer shapes" alt="Bing Maps in Silverlight using Layer shapes" src="http://www.bing.com/community/cfs-file.ashx/__key/CommunityServer.Components.UserFiles/00.00.04.40.32.Attached+Files/4621.SLControl2.JPG" /></p>  <p>Download the SDK here: <a href="http://www.microsoft.com/downloads/details.aspx?displaylang=en&amp;FamilyID=beb29d27-6f0c-494f-b028-1e0e3187e830">Bing Maps Silverlight Control SDK</a> and <a href="http://www.microsoft.com/maps/isdk/silverlight/">check out the interactive SDK</a>.</p>  <h2>Facebook .NET SDK</h2>  <p>An updated (actually 3.0) release of the <strong><a href="http://microsoft.com/facebooksdk">Facebook .NET SDK</a></strong> was also released.  Are you a .NET developer writing Facebook applications or desiring to integrate the social network into your applications?  Check out the Facebook .NET SDK which provides:</p>  <ul>   <li>Silverlight-compatible assembly for handling all the logic to communicate with Facebook platform and has specific support for XAML data binding.</li>    <li>Web API for Facebook IFrame/FBML applications</li>    <li>Using ASP.NET MVC?  There’s an assembly to support that model as well</li> </ul>  <p>The Facebook .NET SDK encapsulates the interaction with the developer platform for Facebook and should help .NET developers in web or client get started in a familiar way with Facebook.  I’m excited to see what folks might do with Silverlight and Facebook!</p>  <p><a href="http://microsoft.com/facebooksdk">Download the Facebook .NET SDK here</a> and get links to getting started with the Facebook API.</p>  <div
