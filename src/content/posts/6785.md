<p>a fix/workaround has been posted to an issue in whidbey that i had <a href="http://timheuer.com/blog/archive/2005/11/06/5761.aspx">previously mentioned dealing with the web.config intellisense being lost</a> when you use the web site administration tool in ASP.NET 2.0 applications.</p>
<p>the workaround (<a href="http://lab.msdn.microsoft.com/ProductFeedback/ViewWorkaround.aspx?FeedbackID=FDBK39924#2">Fix Web Site Administration Tool</a>) basically has you modify a config value in the web admin files.&nbsp; just a reminder to keep that change handy as any updates to the runtime would likely overwrite it until the fix is permanent in whidbey service packs.</p>
<p>thanks to whomever posted that!</p>
