i recently hit a weird snag when i went to change a sample application from using <a href="http://dev.live.com/virtualearth/sdk" rel="tag">virtual earth</a> version 3 to version 4. <p>with doing nothing at all but changing the "v3" text in the virtual earth control reference to "v4" i suddenly got this error message:</p> <blockquote> <p>"Sys.InvalidOperationException: Object Microsoft already exists and is not a namespace.”</p></blockquote> <p>my VE control reference was in my &lt;head&gt; tag, and as soon as I moved it within the body tag, the error went away and all is well.  so there you have it -- if you are getting this error of course :-)</p> <div class="wlWriterSmartContent" id="0767317B-992E-4b12-91E0-4F059A8CECA8:5648bef9-a8a8-4126-999f-46ef8de926ea" contenteditable="false" style="padding-right: 0px; display: inline; padding-left: 0px; padding-bottom: 0px; margin: 0px; padding-top: 0px"></div>
