<P>our friends at apple are starting to bundle more.</P>
<P>i visited a site that had a .mov file on it and for some reason i didn't have quicktime on my machine.&nbsp; could it be that annoying qttask.exe that is always running perhaps that made me remove that thing?</P>
<P>at any rate, i went to download it again and noticed that quicktime 7 is now bundled into itunes6.&nbsp; interesting, i thought.&nbsp; i don't want itunes, i want quicktime.&nbsp; i'm a fan of bundling when it makes sense...but this one didn't to me -- if apple has portions of it site that show interactive media and use a quicktime movie format, then they should supply a quicktime plugin only -- i don't need itunes to display an embedded MOV file on a web page!</P>
<P>but i did find the hidden page: <A href="http://www.apple.com/quicktime/download/standalone.html">quicktime standalone</A></P>
