<p>the leaders of the <a rel="tag" href="http://www.azgroups.com">arizona .net user group</a> have convinced <a rel="tag" href="http://weblogs.asp.net/scottgu"><strong>scott guthrie</strong></a> that phoenix is nice in march (and it is...spring training baby!) and he should visit.  they've organized a super user group meeting and you can find <a href="http://upcoming.yahoo.com/event/414413/">information on when/where here</a>.  it will be hosted at scottsdale center for the arts, which is a pretty killer facility.  i'd recommend carpooling and not messing with the no parking signs (those scottsdale folks like to watch those meters).</p>
<p>having <strong><a title="" href="http://www.scottgu.com">ScottGu</a></strong> come to phoenix immediately after <a href="http://www.visitmix.com">MIX08</a> is a great thing for the community.  if you couldn't make it to MIX, you'll be able to see some new things from scott in what no doubt will be an exciting, new and learning experience.  there are some great things coming out of his group at MIX and this event in phoenix is a must see.  i'm sure he'll cover some great new web stuff, updates on <strong><a rel="tag" href="http://silverlight.net/">silverlight</a></strong> and we'll all be enlightened by how he still manages to stay close to the code while serving a growing army in the developer division!  be sure to block your calendar now!</p>
<p>i know scott won't be able to cover everything, so we'll ensure that <strong><a rel="tag" href="http://thecodetrip.com">the code trip</a></strong> will cover some of the gaps.  be sure to <a href="http://feeds.thecodetrip.com/thecodetrip">subscribe to the feed</a> to know when we'll be coming back through phoenix -- we'll be updating it soon!</p>
