<p><a href="http://thecodetrip.com"><img style="MARGIN: 0px 10px 0px 0px" alt="The Code Trip Logo" align="left" border="0" src="http://s3.amazonaws.com/timheuer-img/codetrip-sign-144.png" /></a>Wow, it's been a <a href="http://timheuer.com/blog/archive/2007/08/13/silverlight-silvertour-road-trip-silverlight-across-america.aspx">few months now since I thought about doing a road trip</a> talking about the next wave of technologies.  I originally thought it would be the "Silvertour" but we've now actually made it happen.  I can tell you that the behind the scenes of this has been a long process.  It seems so simple and I can hear the people now saying 'why was it so hard, c'mon you are Microsoft and have zillions of dollars.'  Sure, maybe that is true about a big company, but that's also the point.  We are a company of companies and thus we still have budgets.  For me, that's been the biggest challenge wrangling in budgets, efforts, desires, balancing other goals, etc.  It's been a long process.</p>
<p>But it's on baby.  <strong><a href="http://thecodetrip.com">The Code Trip</a> </strong>is coming...like in a few days literally.  We thought small (budgets) and wanted to go the RV route.  Well, for various reasons (stop by an event and we can chat), we got a friggin tour bus.  And in fact it's the tour bus coming off the Styx tour.  How crazy is that?  Here's a preview of what we'll be cruising around the western U.S. in:</p>
<p><img alt="The Code Trip Bus" src="http://s3.amazonaws.com/codetrip/buswrap.png" /></p>
