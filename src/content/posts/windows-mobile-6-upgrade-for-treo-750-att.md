<p>as of today 04-DEC-2007, the treo 750 in the US is upgradable by the carrier (AT&amp;T) to windows mobile 6.</p>
<p>while this update has been available for UK roms for a while via vodafone, it is now 'officially' available and supported by at&amp;t.  if you have a treo 750, go <a href="http://www.palm.com/750updateATT">get the update from here</a>.</p>
