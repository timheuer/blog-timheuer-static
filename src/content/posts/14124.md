<p><strong>UPDATE</strong>: okay, so i dug a little deeper, it looks like these are implementing on a windows form host, so your mileage may vary.  getting interaction with other silverlight DOM/canvas objects might be tricky...</p> <p>well, i have to admit that i only spent about the last 2 minutes looking at these, so this might be an irresponsible post :-), but what i saw was impressive.</p> <p><a href="http://www.netikatech.com/">netikatech</a> just announced on their site that they are releasing 40 controls for <a href="http://silverlight.net" rel="tag">silverlight</a> for free!  the product, called "GOA Winforms" describes itself as an implementation of the standard System.Windows.Form class library for rich internet controls...these controls include:</p> <ul> <ul> <li>Control, ContainerControl, ScrollableControl, Panel  </li><li>Button, CheckBox, RadioButton, GroupBox, Label  </li><li>TextBox, NumericUpDown  </li><li>ImageBox, ImageList  </li><li>ScrollBar, HScrollBar, VScrollBar  </li><li>Form, MessageBox, Cursor  </li><li>ListBox, CheckedListBox  </li><li>ComboBox  </li><li>TreeView  </li><li>MonthCalendar  </li><li>TabControl, Splitter  </li><li>ToolTip, ProgressBar, Timer  </li><li>ToolStrip, StatusStrip, MenuStrip, ToolStripButton, ToolStripComboBox, ToolStripDropDown, ToolStripLabel, ToolStripProgressBar, ToolStripSeparator, ToolStripSplitButton, ToolStripTextBox  </li><li>XamlCanvas (Silverlight specific)</li></ul></ul> <p>i'm in the process of installing right now (you must have visual studio 2008 beta 1 to install :-( ), but you can get them by <a href="http://www.netikatech.com/products/default.aspx">downloading here</a>.</p> <div class="wlWriterSmartContent" id="scid:0767317B-992E-4b12-91E0-4F059A8CECA8:7b9a1e9f-8067-4e18-b6b5-703e2b8e9d54" style="padding-right: 0px; display: inline; padding-left: 0px; padding-bottom: 0px; margin: 0px; padding-top: 0px"></div>
