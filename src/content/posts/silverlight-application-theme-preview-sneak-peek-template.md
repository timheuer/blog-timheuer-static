<p><em>‘Twas the week before MIX, when all through the tubes     <br />
Not a developer was sleeping, not even the noobs.</em></p>
<p><em>The laptops were paved removed of their glitz     <br />
In hopes that they soon will get some new bits.</em></p>
<p><em>A developer was coding, building an app     <br />
Trying to build the next greatest XAP</em></p>
<p><em>Battleship gray?! Now that’s obscene     <br />
Check our designers’ latest theme</em></p>
<p>Okay, so I’m not going to win any poetry awards.  Our UX design team for <a href="http://silverlight.net">Silverlight</a> has been thinking about app building a lot this past year, gathering valuable input from developers, designers and end-users about how people interact with applications, primarily <em>line-of-business</em> applications (&lt;shudder&gt;I hate that term&lt;/shudder&gt;).  Hot off the press here is a preview of some of the things we’ve been thinking about from a XAML theme perspective.</p>
<p>First, I present to you codename <em>Grayscale</em>.  Some subtle twists on existing base themes but not detracting too much from the ‘traditional’.  (<a href="http://www.flickr.com/photos/timheuer/4426095286/sizes/o/">larger view here</a>).</p>
<p><img src="http://storage.timheuer.com/grayscale-sampleapp.jpg" alt="Grayscale Silverlight Theme" title="Grayscale Silverlight Theme" style="margin: 0px auto; display: block; float: none;" /></p>
<p>Up next is codename <em>Windows Theme</em> (yeah, original I know, gimme a break here I’m making these up).  Taking a cue from Windows 7 system design, this theme brings familiarity to the end-user. (<a href="http://www.flickr.com/photos/timheuer/4425329927/sizes/o/">larger view here</a>).</p>
<p><img src="http://storage.timheuer.com/wintheme-sample-app.jpg" alt="Windows Silverlight Theme" title="Windows Silverlight Theme" style="margin: 0px auto; display: block; float: none;" /></p>
<p>And finally, <em>Metro</em>.  Taking a cue perhaps from Zune desktop (and device) software design, a clean but fun theme for any application (<a href="http://www.flickr.com/photos/timheuer/4426095220/sizes/o/">larger view here</a>).</p>
<p><img src="http://storage.timheuer.com/white-sample-app.jpg" alt="Metro Silverlight Theme" title="Metro Silverlight Theme" style="margin: 0px auto; display: block; float: none;" />    <br />
So there you have some preview of some Silverlight application themes we’ve been playing around with.  I know the design team is enthusiastic about getting these in the hands of developers/designers.  </p>
<p>What do you think?</p>
