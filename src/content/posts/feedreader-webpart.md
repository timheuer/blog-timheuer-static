<p><a href="http://www.codeplex.com/FeedReader">FeedReader</a> is a web part for Microsoft SharePoint server products (MOSS and WSS).  It’s purpose is to aggregate more than one feed in a single web part.  The built-in XML and RSS web parts for SharePoint only allow one feed by default.  <a href="http://www.codeplex.com/Wiki/View.aspx?ProjectName=FeedReader" title="">feedreader</a> can support Atom or RSS feeds.  Please report any issues as a Work Item on the project.  Source code is also available on the <a href="http://codeplex.com/feedreader">project site</a> under a very permissive license.</p>
