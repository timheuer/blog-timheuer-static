<p>well, for those that have been emailing, i hope you stayed subscribed.  <a title="" href="http://www.timheuer.com/downloads/rssfeedreadersetup.msi">feedreader</a> has finally found a home and the complete source code is available.  </p>
<p>as i’m sure most of you have reflected the code ;-) you’ll know it isn’t brain surgery, but that there are probably some things that could be cleaned up and we know there are areas that need to be fixed.</p>
<p>so visit the <a href="http://www.codeplex.com/Wiki/View.aspx?ProjectName=FeedReader">feedreader codeplex</a> today and get the source released under the <a href="http://creativecommons.org/licenses/by-sa/2.5/">creative commons attribution-sharealike 2.5 license</a>.</p>
<p>remember, the purpose of open source is to share :-D, so be sure to contribute back your fixes and log issues as well!</p>
