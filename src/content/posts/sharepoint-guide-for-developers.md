<p>In February, the <strong>SharePoint</strong> team released a <a href="http://www.microsoft.com/downloads/details.aspx?familyid=3E1DCCCD-1CCA-433A-BB4D-97B96BF7AB63&amp;displaylang=en">Visual Studio extensions kit for SharePoint</a> development.  Just recently they released a user guide complete with samples and walkthroughs.  The sections include:</p>
<ul>
    <ul>
        <li>Starting out in SharePoint Development </li>
        <li>Walkthrough of the VS extensions </li>
        <li>Team Site project </li>
        <li>Blank Site project </li>
        <li>List Definition project </li>
        <li>Web Part project </li>
        <li>Workflow projects </li>
        <li>Project Item Templates </li>
        <li>Best practices for the VS extensions </li>
    </ul>
</ul>
<p>They also note that a VS2008 extensions release looks to be targeted for June 2008.  <strong>Paul Andrew</strong> has all the details with some snapshots on his <a href="http://blogs.msdn.com/pandrew/archive/2008/05/12/announcing-visual-studio-extensions-for-sharepoint-developer-user-guide.aspx">blog post</a> as well.</p>
