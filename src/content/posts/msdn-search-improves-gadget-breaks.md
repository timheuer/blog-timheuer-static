<p>Ah, the joys of using non-API APIs :-).  I say that in jest because essentially <a href="http://www.timheuer.com/blog/archive/2006/12/05/13936.aspx">my first stab at the MSDN search gadget</a> wasn't using any *real* API, but rather just formatting queries correctly and hoping that the format would stay the same.</p>
<p>Well, when you do this you run the risk of things changing.  For <strong>MSDN</strong> (and <strong>TechNet</strong>) search, things have changed 3 times now.  <a href="http://blogs.msdn.com/mikeormond/archive/2008/02/25/msdn-search-gadget.aspx">Mike Ormond points out</a> that another change breaks the default MSDN <strong>gadget</strong> that I had created.  He's posted a fix that works for him (until the next change) and should for others.</p>
<p>Thanks Mike for posting the fix!</p>
