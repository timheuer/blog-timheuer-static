<p><a href="http://www.flickr4writer.com">Flickr4Writer</a> is a plug-in for <a href="http://writer.live.com">Windows Live Writer</a>, a blog authoring tool available free to anyone.  The plug-in enables a Flickr user (Flickr account is required) to browse their own photos as well as others and insert the content into a blog post using Writer.</p>  <p><img alt="" src="http://storage.timheuer.com/flickr4writer.png" /></p>  <p>The plug-in enables you to search all your content, photosets, or search by tags.  It is available for free and can be <a href="http://codeplex.com/Flickr4Writer">downloaded from Codeplex</a>.  For developers, the code is also made available if you want to contribute fixes and features (that comply with Flickr API terms of use).  </p>  <p><strong><font color="#ff0000">UPDATE: Plugin has been updated to follow new Flickr API terms of use.  <a href="http://timheuer.com/blog/archive/2014/06/29/flickr4writer-windows-live-writer-plugin-updated.aspx">Get the latest version here</a>.</font></strong></p>  <p>I hope that you enjoy the plug-in!</p>
