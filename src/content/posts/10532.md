<p>i just got a &lsquo;a new version is available&rsquo; when i logged into my <strike>msn</strike> windows live messenger.</p>
<p>so being the beta monkey i am, i installed it and was immediately reminded of my pet peeves of installers:</p>
<ul>
<li>PROVIDE AN UNINSTALL LINK (not the case here, but always like to make that point &mdash; and no, the add/remove program doesn&rsquo;t count &mdash; it&rsquo;s not mother-in-law proof)</li>
<li>don&rsquo;t install a shortcut to</li>
<ul>
<li>my main start menu (program group okay)</li>
<li>my quickstart toolbar</li>
<li>my desktop</li>
<li>THESE ARE ALL MY SPACES &mdash; leave them alone &mdash; i&rsquo;ll organize what i think is important, don&rsquo;t tell me what/where you think things are important</li></ul>
<li>PRESERVE MY SETTINGS &ndash; installing messenger live didn&rsquo;t preserve my settings/preferences &mdash; it popped up the login screen upon restart (which i turned off) and when i logged in it shows some friggin today screen (which i turned off)</li></ul>
<p>for crying out loud people &mdash; listen to the user &mdash; if i already had it installed, install it the same way.&nbsp; if i didn&rsquo;t, let ME choose how to install (and not through a custom option &mdash; give me all the options up front).</p>
<p>argh.</p>
