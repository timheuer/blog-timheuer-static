<p>One area where <a href="http://silverlight.net">Silverlight</a> can plat an interesting role for your web applications is wherever file uploading to your server needs to be done.  This may sound odd to think of it that way, but if you’ve ever done file upload in web apps (for larger sizes, chunking, etc.) sometimes it is no fun and involves a lot of Javascript.  There is a <strong><a href="http://silverlight.net/learn/learnvideo.aspx?video=69793">video demonstrating on how you can do file upload with Silverlight 2</a></strong> and also helps demonstrate the OpenFileDialog API within the runtime.</p>
<p>I remember seeing a <a href="http://advertboy.wordpress.com/2007/12/20/flickr-multi-picture-uploader-done-in-silverlight/">multi-file uploader that Jose Farado had started</a> back in Silverlight 1.1 days.  It followed the UI inspiration from the Flickr picture uploader and I really liked it.  Jose has been real busy since then and hasn’t had much time to update his older 1.1 samples to Silverlight 2, but a few others have started to emerge.</p>
<p>Over at the <strong><a href="http://silverlight.net/community/gallerydetail.aspx?cat=6">Silverlight community gallery</a></strong>, you can find a bunch of user-submitted content that includes full solutions, samples, helpful tips, etc.  Right now you can find pointers to 2 solutions implementing multi-file uploading using Silverlight as a UI and controller in the process.</p>
<p>One is from <a href="http://www.michielpost.nl/Silverlight/MultiFileUploader/">Michiel Post</a> and the other is from <a href="http://sandbox.inetsolution.com/fileuploadwebsite/FileUploaderTestPage.aspx">InetSolution, Inc</a>.  Both of them provide similar functionality.  (NOTE: none provide full source code at the moment or have alluded that they will.)  One thing that the team at InetSolution did after receiving some inquiries about their work was to write a bit about it in an <a href="http://turnleft.inetsolution.com/2008/08/file_uploader_demo_under_the_h.html">‘under the hood’ post</a> about their effort so far.</p>
<p><a href="http://www.codeplex.com/SLFileUpload"><span style="font-weight: bold;">UPDATE: Michiel Post has now updated the control and provided source on Codeplex.</span></a><br />
</p>
<p>I’d love to see either of these adapt the UI that Jose had implemented…but in the meantime, check them out!</p>
