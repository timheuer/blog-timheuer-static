<p>With the <a href="http://go.microsoft.com/fwlink/?LinkID=143571"><strong>Silverlight 3 Tools for Visual Studio</strong></a>, developers get access to the <em>Silverlight Navigation Application</em> template.  By default the styles (which are located in the project’s <em>Assets/Styles.xaml</em> file) are really good and neutral for the application layout.  Now, keep in mind this is an overall application layout style and not control styles/templates.</p>
<p>In addition to the default look provided by the tools out of the box with <a href="http://silverlight.net/"><strong>Silverlight</strong></a><strong> 3</strong>, we’re providing 7 additional application template themes for you to enjoy, extend, whatever.  <a href="http://blogs.msdn.com/corrinab">Corrina</a> and team had developed these for the community to consume and we’ve put them up on the <a href="http://gallery.expression.microsoft.com">Expression Community Gallery</a> for download.  Here’s a preview with the links to each template:</p>
