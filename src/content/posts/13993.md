<p>i've got a few questions a while back on what i use for screenshots for my blog as well as for other areas (documentation, etc.).  i know there are a lot of popular programs out there, namely snagit, but i settled on a recommendation from scott hanselman and have been using <a href="http://www.ntwind.com/software/winsnap.html" rel="tag">winsnap</a>.</p> <p>sure in vista, there is a snipping tool, but i've not found it to be as efficient as <a href="http://www.ntwind.com/software/winsnap.html">winsnap</a> which is smaller than paint.exe in size!</p> <p>i recently started using the latest version of <a href="http://www.ntwind.com/software/winsnap.html">winsnap</a> and figured i could integrate my other photo tool of choice (<a href="http://www.flickr.com/" rel="tag">flickr</a>) into the mix.  for me, the screenshot thing has always been a multi-step process: take the screenshot, save image, upload to <a href="http://www.flickr.com/" rel="tag">flickr</a>.  well it dawned on me that i could use the same command-line interface that i use with <a href="http://www.flickr.com/tools/" rel="tag">flickruploadr</a> and integrate that into winsnap...and boom, two clicks to heaven.</p> <p>here's what i did...first use <a href="http://www.ntwind.com/software/winsnap.html">winsnap</a> -- it is awesome.  it's light and fast.  it has subtle, yet effective post-shot image manipulation that you can use.  i keep it in my system tray and run on startup.  i also make the default screenshot be a "region" so i can precisely capture what i want.  the latest version also does vista aero glass well.</p> <p>first for <a href="http://www.flickr.com/" rel="tag">flickr</a> you need to install <a href="http://www.flickr.com/tools/" rel="tag">flickruploadr</a> and ensure you authorize the application (read: run it once and upload a picture to your account).  in <a href="http://www.ntwind.com/software/winsnap.html">winsnap</a> go to the tools...customize option.  basically create a command like this:</p> <p><img alt="sshot-3" src="http://static.flickr.com/175/396820018_831e663987.jpg" border="0" /></p> <p>then basically when you take a snapshot, you can change the settings then hit (in my case) ALT+3 and my <a href="http://www.flickr.com/tools/" rel="tag">flickruploadr</a> window comes up and sends directly to <a href="http://www.flickr.com/" rel="tag">flickr</a>.  sweet.</p> <p>if you do a lot of screen captures either for your blog, bugs, or documentation, check out <a href="http://www.ntwind.com/software/winsnap.html" rel="tag">winsnap</a> for sure...it's great.</p> <div class="wlWriterSmartContent"
