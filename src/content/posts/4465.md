<p>so the keynote we&rsquo;ve all been waiting for has finally happened&hellip;this was a great first day in my opinion&hellip;</p>
<p>Highlights:</p>
<ul>
<li>BillG video with Napolean Dynamite &mdash; we saw this at some internal training and despite some parts being cut out, it was still hilarious (unless you have no idea who napolean dynamite is)</li>
<li>Don Box making fun of the memory space VS2005 is using</li></ul>
<p>What we saw:</p>
<ul>
<li>Vista: aero glass effects, sidebar &amp; gadgets, parental controls, peer networking (people near me), stacked documents, virtual folders, etc.</li>
<li>Office12: i&rsquo;ve been waiting to say something about this for a while.&nbsp; what a revolutionary change to some of the core application features in office, i&rsquo;ll be waiting for this release VERY eagerly.&nbsp; <a title="SharePoint" href="http://msdn.microsoft.com/sharepoint" target="_blank">sharepoint</a> is refined (better UI, secure-aware menus, recycle bin) and has a great new feature of the slide library&hellip;</li>
<ul>
<li>slide library allows a ppt to be uploaded and is disected to &ldquo;items&rdquo;</li>
<li>you can then create a new ppt, adding items *from* a sharepoint lib and choose to keep them up-to-date: that&rsquo;s right&hellip;they get changed in SharePoint and so does your powerpoint on your desktop! VERY cool</li></ul>
<li>Outlook12: very cool features for me to help better organize: enhanced &ldquo;to-do&rdquo; listing with tasks built in that roll to next days &mdash; so you&rsquo;re always nagged</li>
<li>InfoPath &mdash; surprising they didn&rsquo;t show some of this stuff in the keynote, and I want to say something about what I like about InfoPath 12, but since I didn&rsquo;t hear anything, I&rsquo;ll wait.</li>
<li>&ldquo;lap around&rdquo; &mdash; Don Box, Chris Anderson, Anders H, ScottGu collaborated on stage and showed:</li>
<ul>
<li>LINQ &ndash; integrated query language &mdash; query objects in code just like you would SQL in a database &mdash; and combine various data across multiple boundaries&hellip;very cool (note: some may have known some of this as parts of Zen)</li>
<li><a href="http://atlas.asp.net/">Atlas</a>: new &ldquo;ajax-style&rdquo; web development framework for asp.net &mdash; cool VirtualEarth control: <a href="http://atlas.asp.net/">go get atlas here</a></li></ul>
<li><a href="http://www.microsoft.com/max">Microsoft Max</a> &mdash; a full featured photo sharing app built using WinFX &mdash; very cool application and available for us all to use&hellip;</li></ul>
<p>more to come&hellip;</p>
