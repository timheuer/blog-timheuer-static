<p>silverlight rc1 also includes a 'refresh' to the alpha version (v1.1) of silverlight, so you'll want to grab that if you are doing the v1.1 development.  because there are breaking changes between beta and this release candidate, samples/sites you may have visited may not work as expected.  in fact, you may have the latest plugin installed (which is now self-updating btw), but go to a site and see the "install beta" image.  if you see this (and know you have the latest plugin installed) it is likely because the author hasn't updated their site yet.  you should contact the author/site...tell them to contact me, i'll help them get it updated and understand the breaking changes.</p>
