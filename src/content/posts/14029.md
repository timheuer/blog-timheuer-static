<p>one thing that i really have been liking about microsoft over the past year is the teams that have been listening to feedback.  here's one case in point.</p> <p><img style="margin: 0px 15px 0px 0px" height="213" alt="Expression Blend" src="http://www.microsoft.com/Expression/images/ex-blend/boxShot_Blend.jpg" width="312" align="left" /></p> <p>previously it was discussed (i'm not sure if was even officially 'announced' or not announced, but there was some discussion at best) that the expression tools (<a href="http://www.microsoft.com/Expression/products/overview.aspx?key=web" rel="tag">expression web</a>, <a href="http://www.microsoft.com/Expression/products/overview.aspx?key=blend" rel="tag">blend</a>, <a href="http://www.microsoft.com/Expression/products/overview.aspx?key=design" rel="tag">design</a>, and <a href="http://www.microsoft.com/Expression/products/overview.aspx?key=web" rel="tag">media</a>) would not be available to <a href="http://msdn2.microsoft.com/en-us/vstudio/aa718657.aspx">premium visual studio subscribers</a> (also known as <a href="http://msdn2.microsoft.com/en-us/vstudio/aa718657.aspx">'msdn subscriptions'</a>).  this caused quite a bit of uproar in the community (and candidly i'll share with you that the uproar internally was ten-fold).  microsoft embarking on this designer/developer story, talking about separation yet cohesiveness via <a href="http://en.wikipedia.org/wiki/XAML" rel="tag">XAML</a>, etc.  you can see where the arguments came from of such a toolset not being made available to purchasers of the premium developer tool suites.</p> <p>well, alas, feedback taken.  much like similar feedback changes, microsoft listened...and reacted.  so today, we see that the expression blend and web tools *will* be available to all premium subscribers of msdn/visual studio.  this is great news!  feedback listened to and changed!  great work community on proving the value of such decisions.</p> <p>what does this mean?</p> <ul> <ul> <li>expression blend and expression web will be made available to <a href="http://msdn2.microsoft.com/en-us/vstudio/aa718657.aspx">msdn premium subscribers</a>  </li><li>expression studio will be made available to team suite w/msdn premium subscribers</li></ul></ul> <p>see the announcements <a href="http://blogs.msdn.com/somasegar/archive/2007/04/03/listening-to-your-feedback-expression-and-msdn.aspx">here</a> and <a
