<p><img align="left" style="margin: 0px 5px 5px 0px; display: inline;" title="Expression Studio 4 box shot" alt="Expression Studio 4 box shot" src="http://www.microsoft.com/expression/fw/studio4-ultimate.png" />Today (7-Jun-2010) at Internet Week in New York, Microsoft announced the general availability of <strong>Expression Studio 4</strong> which includes upgraded versions of <strong>Expression Blend</strong> (including Sketchflow), Encoder, Web (including SuperPreview) and Design.</p>
<p>You can find out the details of each product and <strong>download a trial</strong> at <a href="http://www.microsoft.com/expression">http://www.microsoft.com/expression</a> right now.</p>
<p>With this release comes a free Upgrade for licensed version 3 (Studio or Web) users!  All you need to do is install the trial version of v4 on top of your licensed version of Expression Studio 3 or Expression Web 3 and the installer will find your license and upgrade it to the full v4 product with no expiration.  This applies to customers who received their software through retail channels or electronic software download direct.  For customers who have broader license agreements (i.e., MSDN, WebsiteSpark, BizSpark) you should install the product using the software provided from your program site.</p>
<p>Here’s a quick break-down list of what’s new in this release:</p>
<p><strong>Expression Blend 4</strong> New Features include:</p>
<ul>
    <li>VS2010 compatibility </li>
    <li>Windows Phone support </li>
    <li>Deeper Adobe Photoshop import (layer effects) </li>
    <li>New behaviors &amp; conditional behaviors </li>
    <li>Enhanced sample data support </li>
    <li>Listbox path layout for designing with data </li>
    <li>Pixel Shader effects (including animations) </li>
    <li>Easier styling and customization </li>
    <li>Model View View-Model support </li>
    <li>Mockup controls for SketchFlow </li>
</ul>
<p><strong>Expression Web 4</strong> New Features include</p>
<ul>
    <li>SEO Reporting from inside of the application </li>
    <li>New extensibility model enables creating add-ins with HTML, JS, and CSS </li>
    <li>New SuperPreview online service beta for browser compatibility testing now supports Macintosh Safari </li>
</ul>
<p><strong>Expression Encoder 4</strong> Pro New features include</p>
<ul>
    <li>Live Smooth Streaming (VC-1 &amp; H.264) </li>
    <li>New H.264 encoder from MainConcept </li>
    <li>Enhanced Screen Capture </li>
    <li>DRM (PlayReady) for Live Content </li>
</ul>
<p>This is an awesome release for XAML, web and media developers creating interactive solutions.  </p>
<blockquote>
<p><strong>NOTE</strong>: If you are developing in Silverlight for Windows Phone 7 and need/want to use Blend for this, do not install the released version of Expression Studio 4. You must continue to use the <a href="http://blogs.msdn.com/b/expression/archive/2010/03/15/expression-blend-4-beta-and-information-on-windows-phone-development.aspx">Blend 4 Beta and Add-in Preview for Windows Phone</a>. This Beta will be refreshed with each Phone SDK pre-release and will be unified with released Blend 4 in a service pack which will release when the Windows Phone SDK releases.</p>
</blockquote>
<p>There will be a bunch of information coming out about these features and tutorials by the Expression team along with videos, etc.  I would keep an eye on <a href="http://www.adamkinney.com">Adam Kinney</a>’s site for details on some of this information.  As the Expression Evangelist, Adam is a ‘must subscribe’ resource that you should have in your toolbox!  So go <a href="http://expression.microsoft.com/en-us/ff723798.aspx">check out an overview of Expression Studio 4</a> and get the trial!</p>
