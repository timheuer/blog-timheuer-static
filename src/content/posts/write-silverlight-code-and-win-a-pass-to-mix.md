<p>Did I mention <a href="http://timheuer.com/blog/archive/2009/01/16/register-for-mix09-discount.aspx">I’m going to MIX</a>?  Will I see you there?  I could imagine that some have tight travel budgets, training budgets shrunk, yada, yada.  I’m telling you…if you are a Microsoft developer in the web and WPF space, you need to be at <a href="http://2009.visitmix.com">MIX09</a>.  In an effort to help there are a bunch of ways to reduce some costs…here’s some:</p>  <ul>   <li><a href="https://register.visitmix.com/registrationcreateaccountrsvp.aspx?">Register before 27 Feb 2009 with RSVP</a> code: <strong>MIXLastChance</strong> and get $300 off the registration fee </li>    <li>Share a creative picture about Internet Explorer (<a href="http://blogs.msdn.com/giorgio/archive/2009/02/16/Internet-Explorer-Free-Ticket-Mix.aspx#">details and post yours here</a>) and get a FREE pass to MIX </li>    <li>Tell a fairy tale in WPF or <a href="http://silverlight.net">Silverlight</a> (<a href="http://adamkinney.com/blog/411/default.aspx">details and post yours here</a>) and get a FREE pass to MIX </li>    <li>Solve some riddles on Laurence’s blog (<a href="http://blogs.msdn.com/webnext/archive/2009/02/13/think-you-have-search-engine-skillz-prove-it-and-win-a-pass-to-mix.aspx">details here</a>) and get a FREE pass to MIX </li> </ul>  <p>I figured, why can't I have some fun as well?  I asked on <a href="http://twitter.com/timheuer">Twitter</a> for some suggestions and god a few good ideas.  Here’s my ideas: play a game or draw some pictures.</p>  <p><strong><u>Option 1 – design a Line Rider map for MIX</u></strong></p>  <p>One of my favorite Silverlight game implementations that requires some artistic skills is <a href="http://linerider.com/play-line-rider-online">Line Rider</a>.  So here’s the skinny:</p>  <p><strong>Create a Line Rider map for MIX.  </strong>The map <strong>must include elements of the MIX09</strong> art work (logo or <a href="http://2009.visitmix.com/MIXtify/Art.aspx">current art</a>) <strong>as well as some Las Vegas</strong> bling.  You could do a Las Vegas strip scene with some famous landmarks, making Bosch (the Line Rider character) cruise around the New York, New York rollercoaster, up the Paris tower, etc., etc.  Get creative.  Here’s some inspiration: <a href="http://www.youtube.com/watch?v=bcu8ZdJ2dQo">Jagged Peak Adventure</a>.</p>  <p>Post a link to a video of your track as a <em>comment on this blog post prepending the subject with [Line Rider]</em>.  I’ll pick the winner Monday, 23 Feb 2009.  This gives you a while to do it, but at the same time plan accordingly for a trip to Vegas if you win :-).  This free pass is ONLY for registration so you still have to get yourself to Las Vegas plus any lodging you’d require.</p>  <p><strong><u>Option 2 – Visualize Some Data with Silverlight</u></strong></p>  <p>One of the MIX lab projects is <a href="http://www.visitmix.com/lab/descry">Descry</a>.  This is a project in data visualization.  It isn’t a toolkit that you can automatically apply to any data, but a lab project to take an approach on developing infographics which represent different ways to look at data.  So your task would be to get inspiration from Descry and create a data visualization <em>in Silverlight</em> using public data of some sort.  This could be something depressing like the unemployment rates over time in correlation to Presidential administrations or housing market values across your region.  Or it could be something even more interesting like perhaps the visualization of a Porshce design since the automaker’s inception, something with stock data (not a stock chart), etc.  Whatever.  You define it.  But use real information, nothing made up and <em>name the source of your information</em>!</p>  <p>Post a link to your Silverlight project as a <em>comment on this blog post prepending the subject with [Descry]</em>.  I’ll also pick the winner of this one on Monday, 23 Feb 2009 by end of day.  </p>  <p><strong><u>Option 3 – Create a new preloader</u></strong></p>  <p>Inspired by similar thoughts that <a href="http://pagebrooks.com/archive/2009/02/19/custom-loading-screens-in-silverlight.aspx">Page wrote recently</a> which I agree on, I’m adding another option.  Create a new default preloader for Silverlight apps.  Use some wicked vectors, create an interesting animation.  <a href="http://www.smashingmagazine.com/2008/03/13/showcase-of-creative-flash-preloaders/">Get inspired by other designers</a>.  The goal would be to make it generic enough for anyone to use, but also creative.  That’s only a guide though…get crazy on this one.</p>  <p>Post a link to your running preloader as a <em>comment on this blog post prepending the subject with [Preloader]</em>.</p>  <p>For <strike>both</strike> all of these please be sure to use a REAL email address in the comment form – it isn’t made public but will get to me…I need a way to contact you.  If you have questions, leave them here or ping me on <a href="http://twitter.com/timheuer">Twitter</a> as well!</p>  <p>Have fun and good luck!</p>  <div style="padding-bottom: 0px; margin: 0px; padding-left: 0px; padding-right:
