<p>if you are using <a href="http://www.microsoft.com/sharepoint">sharepoint </a>(either portal server or just the windows sharepoint services) and want to list some news feeds on your sites...check out my <a href="http://www.timheuer.com/stuff.htm">rss feed web part for windows sharepoint services</a>.</p>
<p>i wrote a little <a href="http://www.timheuer.com/stuff.htm">web part</a> that lists rss feeds in a single web part view.  some features included: cached results to avoid continuous http requests (configurable duration), proxy server support (except for user auth proxies), description preview, date sorting, etc.</p>
<p>check it out and let me know what you think...given the whidbey stuff, this should work in <a href="http://www.asp.net/whidbey">asp.net whidbey</a> as well!</p>
