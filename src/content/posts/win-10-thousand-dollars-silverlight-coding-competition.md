<p>I was honored to be asked by ComponentArt to participate in a judging panel for their just now released <strong><a href="http://www.componentart.com/community/competition2009/">2009 Summer Silverlight Coding Competition</a></strong><strong>.  </strong>The grand prize is USD $10,000.  </p>
<p><strong>Yes, that’s right: USD $10,000!</strong></p>
<p><strong><a href="http://www.componentart.com/community/competition2009/"><img border="0" style="border: 0px none ; display: block; float: none; margin-left: auto; margin-right: auto;" title="ComponentArt Silverlight Coding Competition" alt="ComponentArt Silverlight Coding Competition" src="http://s3.amazonaws.com:80/storage.timheuer.com/componentartcodingcomp.png" /></a></strong></p>
<strong>  </strong>
<p>ComponentArt is even kicking in 2 runner-up prizes of licensing to their products, which is also a sweet deal!  The contest is simple, and you are not required to use ComponentArt controls…in their words:</p>
<blockquote>
<p><em>Applicants are not required to use ComponentArt's products to enter the competition (or as the legal jargon requires us to say: NO PURCHASE NECESSARY). We'll obviously </em><a href="http://www.componentart.com/community/competition2009/experience.aspx"><em>let you use our products</em></a><em> if you want to, but the purpose of this contest is to increase awareness of Silverlight features and applications already released, so we would like to keep it open to the entire Silverlight developer community. </em></p>
</blockquote>
<p>Be sure to <a href="http://www.componentart.com/community/competition2009/rules.aspx">read the full contest rules</a> for all the details.  There is a number of judging criteria that will be ranked by 9 total judges: myself, <a href="http://blogs.msdn.com/webnext">Laurence Moroney</a>, <a href="http://silverlight.net/blogs/msnow">Mike Snow</a>, <a href="http://weblogs.asp.net/despos">Dino Esposito</a>, <a href="http://johnpapa.net">John Papa</a>, <a href="http://stevesmithblog.com">Steve Smith</a>, <a href="http://componentart.com/blogs/milos">Milos Glisic</a>, <a href="http://componentart.com/blogs/phil">Phil Tucker</a>, and <a href="http://componentart.com/blogs/corey">Corey Cahill</a>.  As you can see, this is a judging panel made up of Microsoft, ComponentArt and community leaders.</p>
<p>Obviously your application needs to be a <a href="http://silverlight.net/" style="font-weight: bold;">Silverlight</a> application, I hope that’s clear.  It must be accessible via a public entry point and if authentication is required, adequate credentials will need to be supplied.  If after reading the rules you have questions, please ensure you contact the email provided in the rules and FAQ for the contest.</p>
<p>What are you waiting for?  $10K up for grabs!!!!</p>
