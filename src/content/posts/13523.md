<p>some important news came out today regarding visual studio 2005 and visual studio version support on vista.  you can read about it on <a href="http://blogs.msdn.com/somasegar/archive/2006/09/26/772250.aspx">soma's blog here</a>, but here are some nutshell bullets:</p> <ul> <li>Visual Studio 2005 SP1 Beta released today - pending feedback over the next few weeks, this is planned to be released out of beta in the next 3-4 months</li> <li>VS2005 SP1 will run on Vista, but may have some compat issues, teams are working together to find those issues and provide a set of fixes beyond SP1.</li> <li>.NET Framework 1.1 and 2.0 applications will work on Vista (runtimes are supported)</li> <li>Visual Studio 2002 and Visual Studio 2003 will not be supported on Vista.</li> <li>VB6 runtime and IDE will be supported on Vista</li></ul> <p>some of this is good news (SP1) and some not-so-happy-face-news (VS2003 on Vista).  start preparing now...</p> <div class="wlWriterSmartContent" id="0767317B-992E-4b12-91E0-4F059A8CECA8:64eeccc2-d033-41f7-b65b-c901bd63cce2" contenteditable="false" style="padding-right: 0px; display: inline; padding-left: 0px; padding-bottom: 0px; margin: 0px; padding-top: 0px"></div>
