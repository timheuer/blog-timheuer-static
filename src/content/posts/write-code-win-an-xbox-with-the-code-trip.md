<p>One of <strong><a href="http://thecodetrip.com">The Code Trip</a></strong> sponsors, <strong><a href="http://thecodetrip.com/1/toolbox/sponsors#innerworkings">InnerWorkings</a></strong>, has teamed up to have a coding challenge for developers.  This is open to everyone to participate.  The winner gets an annual subscription to InnerWorkings as well as an XBOX 360 game console.</p>
<p>If you aren't familiar with InnerWorkings, consider this a shameless plug.  I really do believe in their product as it is a unique learning experience from what we traditionally have (i.e., lab manuals with step-by-step instructions).  They have a wide catalog of topics including <a href="http://www.asp.net">ASP.NET</a>, WCF, <a href="http://windowsclient.net">WPF</a>, <a rel="tag" href="http://silverlight.net">Silverlight</a>, CSS, etc. for organizations to choose from.  You really need to check them out if you haven't before.</p>
<p>All the details are available on <a href="http://thecodetrip.com/1/innerworkings-coding-challenge">The Code Trip web site recent post</a>.  Head over there for information on InnerWorkings and the information on the contest.</p>
