<p>along with the other ‘best practices’ analyzers that have been released (sql, security, etc.), microsoft’s now released one for asp.net 2.0!</p>
<p>pretty cool:</p>
<blockquote>The Best Practice Analyzer ASP.NET (alpha release) is a tool that scans the configuration of an ASP.NET 2.0 application. The tool can scan against three mainline scenarios (hosted environment, production environment, or development environment) and identify problematic configuration settings in the machine.config or web.config files associated with your ASP.NET application. This is an alpha release intended to gain feedback on the tool and the configuration rules included with it.</blockquote>
