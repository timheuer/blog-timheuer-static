<p>the doclib utils 2.0 web parts for windows <a title="SharePoint" href="http://www.microsoft.com/sharepoint" target="_blank">sharepoint</a> services have been released.</p>
<p>these web parts provide some utility value for document libraries, providing a list of pending approvals and checked-out documents across all document libraries within the site the web part is placed on.</p>
<p>this web part if provided as shareware, please read provided license agreement.</p>
<p>you may download them from <a href="http://www.timheuer.com/stuff.htm">here</a>.</p>
