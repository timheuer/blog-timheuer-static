<p>Building on the positive feedback of the previous <a href="http://www.silverlight.net">Silverlight</a> application <a href="http://timheuer.com/blog/archive/2010/05/03/new-silverlight-4-themes-available-for-download.aspx">themes released last month (Cosmopolitan, Accent Color, and Windows)</a> the design team is working on another theme targeting business application developers.  We don’t yet have an ‘official’ name for this one yet (and to mitigate the confusion of internal code names again, I’ll spare you the code name), but I wanted to put up a preview.  We’re turning this theme around FAST and I wanted to throw it out here in an initial iteration for preview and comment.  This is the ‘dark’ version of the theme and there will be a light version as well.  This represents only the core control set, but like the others will include other SDK controls as well.</p>  <p>I think it is important to understand the goals as well here so that you don’t wonder <em>hey, we need more pixel-shader-gradient-creating-twhirly-things-with-datatype-binding-and-cascading-selector-path-listboxes in themes!!!</em>  In this effort we tasked the designers to build upon the platform of the <a href="http://timheuer.com/blog/archive/2010/05/03/new-silverlight-4-themes-available-for-download.aspx">previous themes</a> and gave them these goals:</p>  <ul>   <li>Should be ‘brandable’ </li>    <li>Work with navigation-style Silverlight applications </li>    <li>Easily tweakable </li>    <li>Will be used most likely by developers with little customization </li>    <li>Professional, clean for customer-based line-of-business applications </li>    <li>Modern, expressive </li>    <li>Wicked cool </li> </ul>  <p>It’s hard to mix all of those requirements into something that will uniformly appeal to everyone.  Given that, here’s the iteration step we are at:</p>  <p><a href="http://drop.io/b53nvtw" target="_blank"><img style="display: block; float: none; margin-left: auto; margin-right: auto" title="Silverlight Application Theme" alt="Silverlight Application Theme" src="http://storage.timheuer.com/jetpack_notitle.jpg" /></a></p>  <p align="center">(click the above image for a full resolution image to see some detail better)</p>  <p>What do you think?  We’re hoping to also be providing some font-alternatives (working through various licensing now) so that you could use different/new embeddable fonts.  </p>  <p>We are moving quickly on this time around and there is little time for change beyond this week.  If you feel strongly about something, share your feedback here ASAP…the designers are listening.  We can’t (and won’t) promise that what you suggest will be implemented (again, trying to please everyone here) but we do want to hear feedback from all sorts of folks who would consider using these professionally designed application themes and styles.</p>  <p>Leave your comments below! </p>  <div style="padding-bottom: 0px;
