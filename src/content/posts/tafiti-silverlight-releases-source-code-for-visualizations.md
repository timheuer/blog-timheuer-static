<p>remember the <a rel="tag" href="http://www.tafiti.com">tafiti</a> site?  want to know how they did that tree visualization in <a rel="tag" href="http://silverlight.net/">silverlight</a>?  now you can.</p>
<p>the tafiti search site is now a windows live quick app -- and uploaded to codeplex.  you can download, view source, mess around and learn.  from the lie team:</p>
<blockquote>
<p>Today we are releasing the <a href="http://codeplex.com/WLQuickApps">Tafiti Search Visualization source code to CodePlex</a>, which means any developer can download, modify, and resell the code (see <a href="http://www.codeplex.com/WLQuickApps/Project/License.aspx">MS-PL License</a> for all the details).</p>
</blockquote>
<p>very, very cool.  go to the <a href="http://codeplex.com/WLQuickApps">codeplex site and check it out</a> and check out the <a href="http://www.liveside.net/blogs/developer/archive/2007/12/18/tafiti-search-visualization-overview.aspx">overview on LiveSide</a>!</p>
