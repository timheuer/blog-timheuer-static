<P>just a reminder to those creating standard images, that with the release of windows xp sp2 (due to release on automatic updates on monday/tuesday, but available for download now via msdn), use the &#8220;slipstream&#8221; model to embed sp2 into your new boot cd images...easy steps:</P>
<P>tools needed:</P>
<UL>
<LI>IsoBuster</LI>
<LI>Nero (or some other tool capable of creating bootable cds)</LI></UL>
<P>steps:</P>
<OL>
<LI>use IsoBuster to extract the boot image from your Windows XP cd-rom (or iso image)</LI>
<LI>copy Windows XP files into a folder on your hard drive (i.e., c:\winxp)</LI>
<LI>run this command from your sp2 setup location: xpsp2.exe /S:C:\winxp</LI>
<LI>the setup will install in &#8220;updating windows share mode&#8220;</LI>
<LI>use Nero or some other tool to create a bootable cd-rom using the image from step 1.</LI>
<LI>label the volume the same label as your original media</LI>
<LI>copy the files from your c:\winxp location to your cd-rom project and burn</LI>
<LI>NOTE: you'll know if the slipstream was successful if in the winxp directory where you have your original media files there is a file named Win51IP.SP2 in there.</LI></OL>
<P>now you have a bootable setup cd with sp2 already embedded in the setup!</P>
