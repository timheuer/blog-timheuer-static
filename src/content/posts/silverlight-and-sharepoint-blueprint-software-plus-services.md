<p>I've gotten a few requests from internal local sales teams, <strong>Sharepoint</strong> professionals and developers about how to implement <strong><a href="http://silverlight.net/">Silverlight</a></strong> in Sharepoint.  I usually try to explain that since Sharepoint is essentially an ASP.NET application it is relatively simple to implement.  Of course, that usually doesn't help :-).  What helps is some documentation and samples that show how you would <strong>use</strong> Silverlight to enhance a Sharepoint experience.</p>
<p>As a part of the <strong>Software+Services Blueprints</strong> series, a <a href="http://msdn2.microsoft.com/en-us/sharepoint/cc303301.aspx">new bundle has been made available to download </a>to provide guidance <strong>and source code</strong> to help provide some information.  In the blueprint you'll get:</p>
<ul>
    <ul>
        <li>A simple “hello, world” sample showing Silverlight in a Web part. </li>
        <li>A Silverlight slider control as a SharePoint custom field type. </li>
        <li>A navigation control. </li>
        <li>A Colleague Viewer that uses lookups in Active Directory by using <strong>Windows Communication Foundation</strong>. </li>
        <li>A Silverlight picture viewer from a SharePoint picture library. </li>
        <li>A visual how-to center, created in Silverlight and based on a SharePoint list, for viewing videos. </li>
    </ul>
</ul>
<p>A pretty great resource for those wanting to discover implementation details as well as some ideas for Silverlight in Sharepoint!</p>
