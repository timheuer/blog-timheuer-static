<p>the windows live writer team has <a href="http://writerdevzone.spaces.live.com/blog/cns!FF912D98C958E9D3!106.entry">just announced</a> that they will be supporting a new section in the <a href="http://gallery.live.com" rel="tag">windows live gallery</a> for writer plugins.  this is a great opportunity to get your plugins ready for primetime and ensure that you are listed in the directory.  <a href="http://jeftek.spaces.live.com/blog/cns!F2042DC08607EF2!610.entry">jeff has been doing a good job keeping track</a>, but now he won't have to worry about it :-).</p> <p>submissions aren't automatic, so be sure to review the criteria.  namely, you need to ensure that you are putting them in an installer package (apparently the preferred is an MSI - <a href="http://timheuer.com/blog/archive/2006/08/25/13364.aspx">more on my opinion here</a>) and bootstrapping any .NET 2.0 installation if needed.</p> <p>of course with submitting things to microsoft properties, there are things you have to agree to, so be sure to read the guidelines appropriately.  this is a pre-approval process...so be one of the first to be on the gallery!</p> <p>read the announcement here: <a href="http://writerdevzone.spaces.live.com/blog/cns!FF912D98C958E9D3!106.entry" rel="tag">Writer Plugins Gallery</a></p> <div class="wlWriterSmartContent" id="0767317B-992E-4b12-91E0-4F059A8CECA8:14136d48-fb07-4b16-9d6f-3ea0bf5588bd" contenteditable="false" style="padding-right: 0px; display: inline; padding-left: 0px; padding-bottom: 0px; margin: 0px; padding-top: 0px"></div>
