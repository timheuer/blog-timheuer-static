<P>famed wss expert <A href="http://mikeswss.blogg.de/">mike walsh</A> had a chance to post about <A title="" HREF="/downloads/rssfeedreadersetup.msi">feedreader</A>, you should check out his <A href="http://mikeswss.blogg.de/eintrag.php?id=32">comments</A>.&nbsp; one thing he points out that is interesting is the last comment about writing competitive freeware products when free ones already exist.&nbsp; it is an interesting point to ponder...like, how many different types of plain text editors can you have and are the feature gaps enough to warrant saying &#8220;it's better&#8221;?</P>
<P>at any rate, i've seen a bunch of rss web parts come out recently for <A title=SharePoint href="http://www.microsoft.com/sharepoint" target=_blank>sharepoint</A> and thought i'd note on some of them.&nbsp; most of them seem to be a derivitive of the xml web part that comes with <A title=SharePoint href="http://www.microsoft.com/sharepoint" target=_blank>sharepoint</A>...a wise idea since rss is an xml format anyway...simply apply an xsl stylesheet and you are good to go.</P>
<P>but let's say you can't spell xsl...</P>
<P>my web part (to toot my own horn here...the first public web part for rss and <A title=SharePoint href="http://www.microsoft.com/sharepoint" target=_blank>sharepoint</A> services) adds value in several areas.&nbsp; first, xml web part only allows a single xml document (either url or text)--and some other rss web parts also have this limitation.&nbsp; <A title="" HREF="/downloads/rssfeedreadersetup.msi">feedreader</A> allows multiple in one web part with no additional configuration.&nbsp; <A title="" HREF="/downloads/rssfeedreadersetup.msi">feedreader</A> also provides value by adding caching capabilities (do you really want a request to go out every time the page refreshes?!?!), item filtering by date and counts, collapse/expand capabilities to read the summary comments, initial atom 0.3 feed support [note: <A title="" HREF="/blog/articles/390.aspx">see known issues on feedreader</A>].&nbsp; the xml web part can't offer you these (i would argue can't, but i'll at least say &#8220;can't easily offer&#8221;) out of the box.&nbsp; <A title="" HREF="/downloads/rssfeedreadersetup.msi">feedreader</A> can...and guess what, it's free too.</P>
<P>thanks for the notes mike...i appreciate it.&nbsp; i also appreciate competition...it makes me strive to see what i can do better (which is the main reason i've been holding off the workspace view web part...another one that i released and many more followed, some good and some not so good).</P>
