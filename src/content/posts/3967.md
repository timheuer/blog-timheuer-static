<p>as noted in <a href="http://blogs.msdn.com/somasegar/">somasegar</a>&rsquo;s web log, <a href="http://blogs.msdn.com/somasegar/archive/2005/08/22/451026.aspx">visual studio 2005 will see another public drop</a> of the visual studio 2005 product lines.&nbsp; some important notes to point out:</p>
<ul>
<li>Visual Studio 2005 will release an RC1 in September, available to MSDN subscribers, early adopters already identified, and beta customers.</li>
<li>Team Foundation Server (TFS), the core collaboration, source control management, etc. for the Team System product lines will have a Beta3 release, including Go Live licensing support.</li>
<li>TFS will ship within the first quarter of 2006</li></ul>
