<p>sure enough as i told the <a href="http://www.denvervisualstudio.net/">denver visual studio user group</a> last night that the only reason i was still running visual studio 2008 beta 2 was because the <a rel="tag" href="http://silverlight.net"><strong>silverlight</strong></a> tools had not been updated yet.  check my email later...they are updated :-).</p>
