<p>Want a <a href="http://silverlight.net/">Silverlight</a>-branded XBOX console?  Yeah, me too, that would be cool.  Well, there isn’t one of those, but there is a chance for you to show your Silverlight prowess and win some cool stuff.</p>  <p><img style="margin: 0px 10px 0px 0px" height="169" src="http://assets.devx.com/ria/xbox.jpg" width="240" align="left" /> The <a href="http://www.devx.com/RIA/Door/35735">RIA development portal</a> at <strong>DevX</strong> is currently <a href="http://www.devx.com/RIA/Door/37507">running a contest</a> (which ends next week, yikes, hurry!) for creating games in <strong>Silverlight 2</strong>.  The contest seems relatively simple:</p>  <p>Step 1: Create wicked cool game in Silverlight 2   <br />Step 2: Host it on Silverlight Streaming</p>  <p>How simple is that…oh yeah, except for the ‘creating game’ part.  There are a few places to help you get started creating games in Silverlight.  Check out the <a href="http://silverlightrocks.com/Community/blogs/silverlight_games_101/">Silverlight Games 101</a> site which has some great posts and pointers to help in creating casual games.  </p>  <p>The prizes are an <strong>XBOX 360</strong> (sorry, not Silverlight branded), an 80GB <strong>Zune</strong> and a game bundle including <strong>Halo 3</strong> and Mass Effect.</p>  <p>So head over to the <a href="http://www.devx.com/RIA/Door/37507">RIA Run site</a> and get all the details to get submitted by <strong>15 MAY</strong>!!!</p>  <div class="wlWriterSmartContent" id="scid:0767317B-992E-4b12-91E0-4F059A8CECA8:f40998a0-8a83-4e49-bac1-16a42ade7816" style="padding-right: 0px; display: inline; padding-left: 0px; float: none; padding-bottom: 0px; margin: 0px; padding-top: 0px"></div>
