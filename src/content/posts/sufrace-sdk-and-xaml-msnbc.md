<p>Saw some posts today over at <a href="http://blog.donburnett.com">Don’s site</a> about Surface.  The Surface SDK is starting to get more visible whereas before it seemed a little black-boxish to me.  Turns out (as we all knew) it really is just WPF with some unique Surface-like behaviors in the SDK.  Take this quick demonstration from a program manager on the SDK.  With using the same concepts that we use in WPF and <a href="http://silverlight.net">Silverlight</a> for data binding, etc. you can come up with a quick application using the inherent gesture support from a Surface device:</p>  <div style="padding-bottom: 0px;
