<p><strong>The source code for the </strong><a href="http://silverlight.net"><strong>Silverlight</strong></a><strong> 2 controls was just published</strong>!  This is the source for the controls as shipped in the runtime and the SDK.  When you download the project you’ll see the structure:</p>  <p><img title="Image of source project tree in Visual Studio" alt="Image of source project tree in Visual Studio" src="http://storage.timheuer.com/sl-control-proj.jpg" /></p>  <p>The source, <strong>released under the </strong><a href="http://www.opensource.org/licenses/ms-pl.html"><strong>Microsoft Public License</strong></a><strong> (</strong><a href="http://www.opensource.org/licenses/ms-pl.html"><strong>Ms-PL</strong></a><strong>)</strong>, includes the source for the following controls in the runtime:</p>  <ul>   <ul>     <li>ButtonBase</li>      <li>Button</li>      <li>HyperlinkButton</li>      <li>CheckBox</li>      <li>RadioButton</li>      <li>ToggleButton</li>      <li>RangeBase</li>      <li>ProgressBar</li>      <li>Slider</li>      <li>ScrollBar</li>      <li>Thumb</li>   </ul> </ul>  <p>and the following from the SDK (part of the Silverlight Tools for Visual Studio):</p>  <ul>   <ul>     <li>Calendar</li>      <li>DatePicker</li>      <li>DataGrid</li>      <li>TabControl</li>      <li>GridSplitter</li>   </ul> </ul>  <p>Unit tests for the SDK controls have also been included as well!</p>  <p>With the release of this source code as well as having access to the source for the <a href="http://codeplex.com/Silverlight">Silverlight Toolkit</a>, you should have some great base implementations to extend and learn from!  Get the code here: <strong><a href="http://www.microsoft.com/downloads/details.aspx?FamilyID=EB83ED4C-AC85-4DE9-8395-285628EE2254&amp;displaylang=en">Source code for Silverlight 2 controls</a></strong>.</p>  <div style="padding-bottom: 0px;
