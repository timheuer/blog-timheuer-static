<p>keynote has started, billg is here in rare fashion: no powerpoint.&nbsp; now clearly there is some powerpoint that *he* is reading from, but taking a step from the <a href="http://www.presentationzen.com/">presentation zen</a>, there is no visual slides that the audience is looking at &mdash; none.&nbsp; none for billg.&nbsp; rare.&nbsp; i think he&rsquo;s doing okay, but definitely getting used to it.&nbsp; because there are no distracting informational slides, more attention is on bill.</p>
<p>the message here is clear: user experience matters.&nbsp; bill&rsquo;s comments are rife with that tone from all aspects of meda: browser, media center, mobility, origami, etc. &mdash; i think he is starting to mix too much technology speak into the user experience message.</p>
<p>he did announce atlas and go-live.&nbsp; the atlas site was refreshed today: <a href="http://atlas.asp.net/">GO VISIT ATLAS</a>.&nbsp; if you want to implement atlas capabilities in an *existing* site, please also look at the install guide and the section that outlines how to do this: <a href="http://atlas.asp.net/docs/overview/install.aspx">INSTALL ATLAS</a>.</p>
<p>aber whitcomb (cto) of <a href="http://www.myspace.com/">myspace</a> is out now.&nbsp; myspace.com has grown to 65 million users, 23 billion page views, and now surpasses msn.com as the #2 most viewed web site according to media metrix.&nbsp; some quick stats: at 9M users, they converted to ASP.NET and saw immediate performance gains; at 17M users, they implemented a dynamic caching engine and achieved 92% cache hit ratio; at 26M users, sql2005 database was implemented.&nbsp; their .net performance gains are pretty remarkable&hellip;CPU usage from 85% to 27%, and a reduction in servers from 246 to 150.</p>
<p>the myspace guys are now doing a demo of their site.&nbsp; the url shows a lot of local file usage, not realtime demo&hellip;kinda disappointing i think, but there are probably reasons.&nbsp; they showed a few new features they will be launching in the future.&nbsp; the term &ldquo;off the browser&rdquo; seems to be hot.&nbsp; myspace is showing a vista gadget they developed to interact with myspace.com features.&nbsp; they also showed a new feature called atlas shuffler or profile 2.0, allowing the user to define what elements are implemented on their personal myspace.</p>
<p>ashley highfield (director of new media and technology) of the bbc is on stage now, showing their vista gadget (again, an &ldquo;off the browser&rdquo; experience) and this is VERY impressive&hellip;a HUGE use of <a href="http://msdn.microsoft.com/winfx/reference/presentation/default.aspx">avalon</a> and all sorts of different features.&nbsp; trying to write it here and describe it definitely doesn&rsquo;t do it service at all&hellip;VERY VERY cool work, great job BBC!&nbsp; almost an &ldquo;itunes-ish&rdquo; view of browsing media and sharing media contents, but the ability to share across users, send to media center, etc.&nbsp; ashley is a pretty funny guy who made some humurous remarks aboout some english videos he is showing.</p>
<p>tim o&rsquo;reilly is now coming out.&nbsp; (he made a coment about safari running on sql 2005/asp as well as &ldquo;having some books with monkeys on them&rdquo;)</p>
<p>tim is aksing bill about the proliferation of the apis, making some good commnets/questions about api accessibility &mdash; pointing out that microsoft has a &ldquo;sales&rdquo; process in getting access to the apis (getting developer key, etc.).</p>
<p>i don&rsquo;t think i&rsquo;ll emit the entire conversation here (i&rsquo;m sure the transcript will be available soon).</p>
