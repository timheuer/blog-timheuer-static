<p>as i continue to build on top of my <a href="http://timheuer.com/blog/archive/2007/01/31/13974.aspx">virtual earth example that uses proximity searching</a>, i found myself wanting to add more visualization to the mix.  i wanted the radius being searched to actually show a radius ring for the area...so the user could see the proximity being searched where the results are plotted.</p> <p>so, i started digging -- how do you draw a circle using the polyline api?  hmmm...rounded corners with straight lines...seemed like a challenge.</p> <p>well after a prod to some people i was pointed to a solution already doing this!</p> <p>and thus the <a href="http://viavirtualearth.com/Wiki/Welcome.ashx" rel="tag">ViaVE Wiki</a> had the answer.  basically it is a javascript function like this:</p> <div class="csharpcode"><pre><span class="lnum">   1:  </span><span class="kwrd">function</span> addCircle(polyname, latin, lonin, radius, vecolor, linewidth) </pre><pre><span class="lnum">   2:  </span>{ </pre><pre><span class="lnum">   3:  </span>    <span class="kwrd">var</span> locs = <span class="kwrd">new</span> Array(); </pre><pre><span class="lnum">   4:  </span>    <span class="kwrd">var</span> lat1 = latin * Math.PI/180.0; </pre><pre><span class="lnum">   5:  </span>    <span class="kwrd">var</span> lon1 = lonin * Math.PI/180.0; </pre><pre><span class="lnum">   6:  </span>    <span class="kwrd">var</span> d = radius/3956; </pre><pre><span class="lnum">   7:  </span>    <span class="kwrd">var</span> x; </pre><pre><span class="lnum">   8:  </span>    <span class="kwrd">for</span> (x = 0; x &lt;= 360; x++) </pre><pre><span class="lnum">   9:  </span>    { </pre><pre><span class="lnum">  10:  </span>        <span class="kwrd">var</span> tc = (x / 90)* Math.PI / 2; </pre><pre><span class="lnum">  11:  </span>        <span class="kwrd">var</span> lat = Math.asin(Math.sin(lat1)*Math.cos(d)+Math.cos(lat1)*Math.sin(d)*Math.cos(tc)); </pre><pre><span class="lnum">  12:  </span>        lat = 180.0 * lat / Math.PI; </pre><pre><span class="lnum">  13:  </span>        <span class="kwrd">var</span> lon; </pre><pre><span class="lnum">  14:  </span>         <span class="kwrd">if</span> (Math.cos(lat1)==0) </pre><pre><span class="lnum">  15:  </span>        { </pre><pre><span class="lnum">  16:  </span>            lon=lonin; <span class="rem">// endpoint a pole </span></pre><pre><span class="lnum">  17:  </span>        } </pre><pre><span class="lnum">  18:  </span>        <span class="kwrd">else</span> </pre><pre><span class="lnum">  19:  </span>        { </pre><pre><span class="lnum">  20:  </span>            lon = ((lon1 - Math.asin(Math.sin(tc) * Math.sin(d)/Math.cos(lat1)) + Math.PI) % (2 * Math.PI)) - Math.PI; </pre><pre><span class="lnum">  21:  </span>        } </pre><pre><span class="lnum">  22:  </span>        lon = 180.0 * lon / Math.PI; </pre><pre><span class="lnum">  23:  </span>        <span class="kwrd">var</span> loc = <span class="kwrd">new</span> VELatLong(lat,lon); </pre><pre><span class="lnum">  24:  </span>        locs.push(loc); </pre><pre><span class="lnum">  25:  </span>    } </pre><pre><span class="lnum">  26:  </span>    <span class="kwrd">var</span> poly = <span class="kwrd">new</span> VEPolyline(polyname, locs, vecolor , linewidth); </pre><pre><span class="lnum">  27:  </span>    <span class="kwrd">return</span> poly; </pre><pre><span class="lnum">  28:  </span>}</pre></div>
