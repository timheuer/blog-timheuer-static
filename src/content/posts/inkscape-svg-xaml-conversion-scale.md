<p><img style="MARGIN: 0px 10px 0px 0px" alt="Inkscape.org" align="left" src="http://s3.amazonaws.com/timheuer-img/inkscape.png" />i had <a href="http://timheuer.com/blog/archive/2007/08/30/convert-vector-svg-to-xaml-for-silverlight.aspx">previously done a screencast</a> where i talked about the tools i use to convert vector graphics in SVG format to XAML assets to be consumed in either <strong>WPF</strong> or <strong><a href="http://silverlight.net/">Silverlight</a></strong> applicaitons.  one of the tools mentioned in the screencast is <strong><a rel="tag" href="http://www.inkscape.org">Inkscape</a></strong>.  inkscape is an open source vector graphics editor, with capabilitis siilar to illustrator, freehand, etc and uses the SVG file format (mostly).  one of the cool things about inkscape is the extensibility already offered in the open source platform.  after having some colorful discussions with my local linux group on their discussion group, i met one of the inkscape contributors.  we 'chatted' (read: emailed) privately off the group for a while about vector formats, why not SVG, etc., etc. and ended up in an amicable place understanding the role of XAML.  through this, ted, helped contribute a checkin to the inkscape project that enables files being edited in inkscape to be exported directly to XAML format.</p>
<p>ted tells me the <a href="http://www.socallinuxexpo.org/scale6x/conference-info/exhibitions/Inkscape/">inkscape guys will have a booth</a> at the <a href="http://www.socallinuxexpo.org/">southern california linux expo</a> in february.  i think that they'll have a stable build of the xaml feature by then (crossing fingers).  if you are brave enough for daily builds it is already in there if you want to try it out.  if you are going to be in in southern california, you should stop by the expo and chat it up with the inkscape crew that will be manning the booth.  it's a great tool that has helped me in the past few months.  while the conference isn't free, well neither is the expo, the expo hall looks like a $10 ticket.</p>
