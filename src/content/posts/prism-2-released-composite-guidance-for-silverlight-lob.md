<p>The big news in the <a href="http://silverlight.net">Silverlight</a> developer world today is the release of Prism v2 (also called the <strong>Composite Application Guidance</strong>).  So what is this?</p>  <p><strong>Prism</strong> guidance is a set of tools, samples, references and written guidance to help you more easily build modular applications.  Generally the “modular” application will feature several screens, flexible user interaction and role-based behavior.  Composite applications using these patterns are meant to be loosely coupled and contain independently evolving pieces that can work together.  So in the <a href="http://www.microsoft.com/downloads/details.aspx?FamilyID=fa07e1ce-ca3f-4b9b-a21b-e3fa10d013dd&amp;DisplayLang=en">Prism 2 release</a> you are provided:</p>  <ul>   <li>Composite Application Library</li>    <li>Reference Implementation (Stock Trader application)</li>    <li>9 Quick starts</li>    <li>26 How-to’s</li>    <li>Documentation and written guidance on the UI patterns and client architectures you may face</li> </ul>  <p>There has been much talk about the <strong>Model-View-ViewModel (MVVM)</strong> pattern for <a href="http://www.windowsclient.net">Windows Presentation Foundation</a> (WPF)and Silverlight development.  The Prism release adapts this model (refers to this as the <em><strong>presentation model</strong> </em>to match what some other pattern documentation in the greater technology world uses) in the reference implementation of the Stock Trader application.</p>  <blockquote>   <p>NOTE: The Stock Trader application is a reference implementation of the composite application guidance.  It isn’t meant to actually server real stock trading, but was inspired by those similar scenarios.</p> </blockquote>  <p>Prism 2 is an evolution from a July 2008 release that was primarily for WPF applications.  This new release brings updates and those concepts to Silverlight, including an implementation of commanding in Silverlight as well as demonstration of the use of input validation using these concepts.</p>  <p>For a walk through of some of the concepts and a brief discussion from the Patterns and Practices program management team, <a href="http://channel9.msdn.com/shows/Continuum/Prismv2/">watch the latest Continuum show about Prism with Blaine Wastell</a>.  This is a great development evolution for line-of-business application developers.  <a href="http://www.microsoft.com/compositewpf">Check it out</a>!</p>  <div style="padding-bottom: 0px; margin: 0px;
