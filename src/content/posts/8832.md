<p>the second session immediately following the billg keynote was about ie7, here&rsquo;s some key messages:</p>
<ul>
<li>css/html compliance &mdash; working hard to comply with core web standards in these areas and more.&nbsp; ie7 actually kinda breaks itself (ie6) when previous hacks were used to &ldquo;fix&rdquo; things in ie6.&nbsp; check out the ie7 developer center on msdn for details on preparing for ie7</li>
<li>ie7 beta 2 preview &ldquo;refresh&rdquo; is available for download at <a href="http://microsoft.com/ie">http://microsoft.com/ie</a></li>
<li>rss &ndash; rss is built in to ie7 and brought to the surface a lot more&hellip;showed rss developer sample of the Feeds API</li>
<li>search providers &mdash; demonstrated how ie7 supports opensearch technologies for search providers.</li></ul>
<p><a title="" href="http://www.scottgu.com" >ScottGu</a> came out and did some great demos on atlas using the latest bits and showing the TODO list sample he has completed.&nbsp; all good stuff and super easy.&nbsp; he also toggled to a mac and showed the app working on safari all good and cool-like.</p>
