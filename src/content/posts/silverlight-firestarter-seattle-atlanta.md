<p>I have the pleasure of being invited to participate in two <a href="http://silverlight.net"><strong>Silverlight</strong></a> events coming up soon (one very soon).  These are a part of a “firestarter” event that is intended to get you familiar with certain technologies, in this case Silverlight, rather quickly and gain a better understanding for the overall platform offerings.  I’m excited to have been invited to participate in both the Atlanta and Seattle events.</p>  <h2>Silverlight Firestarter Atlanta</h2>  <p>The first one will be in Atlanta, Georgia on 22-August!  It’s been quite some time since I’ve been to Atlanta and I’m very happy to be going to hang out with Silverlight developers there.</p>  <p><a href="http://www.silverlightatlanta.net/Firestarter/minisite.html"><img style="border-bottom: 0px; border-left: 0px; display: block; float: none; margin-left: auto; border-top: 0px; margin-right: auto; border-right: 0px" title="Atlanta Silverlight firestarter logo" border="0" alt="Atlanta Silverlight firestarter logo" src="http://s3.amazonaws.com:80/storage.timheuer.com/atl-firestarter.jpg" /></a></p>  <p>The agenda looks to be a good one if you are in the area.  It is a free event to attend and will be hosted at the Microsoft office in Alpharetta.  Apparently it is already booked full for registration.  You can find out more information about the plans, location, and schedule at the <a href="http://www.silverlightatlanta.net/Firestarter/minisite.html">minisite</a>.  Be sure to check out the <a href="http://www.silverlightatlanta.net/Firestarter/minisite.html#Bios">Bios</a> section to see the star studded experts that will be there!  <a href="http://www.wildermuth.com">Wildermuth</a> said that he’d take me to the largest aquarium…I’m hoping that’s not a ploy to get me eaten by some giant whale shark or something.  I’ll settle for BBQ if we can’t make it to the aquarium Shawn!</p>  <h2>Silverlight Firestarter Seattle</h2>  <p>Well, technically Redmond, but who’s counting :-).  This event on 17-September will be held at the conference center on the Microsoft campus.  Registration is required, so make sure you <a href="http://msevents.microsoft.com/CUI/EventDetail.aspx?EventID=1032422412&amp;Culture=en-US">register now</a> if you plan on attending.</p>  <p><a href="http://msevents.microsoft.com/CUI/EventDetail.aspx?EventID=1032422412&amp;Culture=en-US"><img style="border-bottom: 0px; border-left: 0px; display: block; float: none; margin-left: auto; border-top: 0px; margin-right: auto; border-right: 0px" title="Seattle Silverlight firestarter logo" border="0" alt="Seattle Silverlight firestarter logo" src="http://s3.amazonaws.com:80/storage.timheuer.com/sea-slfirestarter.jpg" /></a></p>  <p>This is a pretty packed agenda with some great folks, including <a href="http://weblogs.asp.net/scottgu"><strong>Scott Guthrie</strong></a>, <a href="http://blogs.msdn.com/brada"><strong>Brad Abrams</strong></a>, <a href="http://www.adamkinney.com"><strong>Adam Kinney</strong></a>, <a href="http://silverlight.net/blogs/justinangel"><strong>Justin Angel</strong></a> and <a href="http://karlshifflett.wordpress.com"><strong>Karl Shifflett</strong></a>.  I’m honored that Scott would open up for me ;-).  Seriously though, this looks like it will be a good time!  Adam posted his vision on how the day will go.  I better start working on those loads!  Again, <a href="http://msevents.microsoft.com/CUI/EventDetail.aspx?EventID=1032422412&amp;Culture=en-US">registration is required</a>, so make sure you go do that now.</p>  <p>Hope to see you at one of these!</p>  <div style="padding-bottom: 0px;
