<P>sure enough this happens to me too frequently.</P>
<P>"sure, yeah, no problem i can do that"</P>
<P>then reality strikes...oh yeah, i have a job, wife, kids, no time, etc.</P>
<P>so i find myself in another predicament...a project due next week and only 30% complete.&nbsp; so how do you finish 70% in a few days (and actually less in hours)?</P>
<P>jack daniels and water. <EM>oh wait, did i say that outloud?</EM></P>
<P>okay seriously...here's what helped:</P>
<P>1-written out plan<BR>i find that if i don't write a plan down, i'll continue to procrastinate and go all over the place.</P>
<P>2-realize you can't do everything<BR>if you think you can do it all, you can't -- get over it and get help</P>
<P>3-find help<BR>for me it was in the form of short-cuts in technology...these tools saved me:</P>
<UL>
<LI>visual studio .net -- duh. 
<LI>aspNetEmail -- <A href="http://www.aspnetemail.com">www.aspnetemail.com</A> -- the EmailTemplate control rocks 
<LI>LLBLGen Pro -- <A href="http://www.llblgen.com">www.llblgen.com</A> -- one of my new favorite tools -- rapid generation of ORM-based DAL and great object model 
<LI>FreeTextBox -- <A href="http://www.freetextbox.com">www.freetextbox.com</A> -- they should really name it [NotSo]FreeTextBox since you have to pay for the cool features ;-) 
<LI>Enterprise Library -- <A href="http://www.microsoft.com/practices">www.microsoft.com/practices</A> -- with the new enterprise library, the configuration is so much easier to understand and quick to setup 
<LI>MasterPages -- i rolled my own implementation based on the Paul Wilson article -- not having to worry about the elements is awesome...concentrate on the functionality 
<LI>CodeRush -- <A href="http://www.devexpress.com">www.devexpress.com</A> -- only improves vs.net 
<LI>BaseClasses -- implement common functionality into base classes so that controls/master pages can be rapidly implemented 
<LI>WebUI 2.1 -- <A href="http://www.componentart.com">www.componentart.com</A> -- for great menus, site maps, etc.</LI></UL>
<P>these are huge timesavers for me...check them out -- i highly endorse all of them!</P>
<P>4-quiet place with least distractions<BR>probably the most difficult for me -- i'm usually easily distracted at home -- especially with a young one running around.&nbsp; finding some type of peace is key to getting &#8220;in the groove&#8221; -- for me it is in my office, locked door, ipod in ears, and wife away [note: wife away usually means shopping, so this procrastination may ending up actually costing money ;-) ]</P>
