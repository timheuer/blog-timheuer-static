   <li>xamWebTimeline </li>    <li>xamWebToolbar </li> </ul>  <p>These are some really great visualizers and they have interesting samples of all of them in their <a href="#top">(back to top)</a></p>  <h2><a name="slcontrib"></a><a href="http://silverlightcontrib.org">SilverlightContrib</a></h2>  <p>This is an Open Source project being driven by the community and Silverlight MVPs, namely <a href="http://pagebrooks.com">Page Brooks</a>.  This group of community folks have rallied to create some controls and framework libraries.  Their current controls include:</p>  <ul>   <li>Color Picker </li>    <li>Gauge Control </li>    <li>Star Selector </li>    <li>Enhanced Metafile </li>    <li>Cool Menu </li>    <li>Zip, Byte, String Utilities </li>    <li>Animation Tweening </li>    <li>Wheel mouse listener </li> </ul>  <p>This is great to see this community-driven type resources.  Their demo application is also available for <a href="http://silverlightcontrib.org/demo/">perusing here</a> and you can download the bits/code on their <a href="http://codeplex.com/silverlightcontrib">CodePlex project site</a>.</p>  <p><a href="#top">(back to top)</a></p>  <h2><a name="toolkit"></a><a href="http://codeplex.com/Silverlight">Silverlight Toolkit</a></h2>  <p>Who could forget the beloved Silverlight Toolkit!  Coming from a team inside Microsoft, this is an Open Source project led by a team within Microsoft to provide a set of controls with full source code to developers to leverage, extend, etc.  The controls are:</p>  <ul>   <li>AutoCompleteBox </li>    <li>DockPanel </li>    <li>HeaderedContentControl </li>    <li>HeaderedItemsControl </li>    <li>Label </li>    <li>NumericUpDown </li>    <li>TreeView </li>    <li>WrapPanel </li>    <li>Charting </li>    <li>Expander </li>    <li>ImplicitStyleManager </li>    <li>Viewbox </li>    <li>Set of XAML themes </li> </ul>  <p>The goal of this project is to provide some extended controls beyond the core that Silverlight provides with the assumption that some of the controls from this project <em>may</em> make it into future releases of Silverlight’s core framework.  You can view the project and sample applications on the <a href="http://www.codeplex.com/Silverlight">Silverlight Toolkit CodePlex project site</a>.</p>  <p><a href="#top">(back to top)</a></p>  <h2><a name="slext"></a><a href="http://www.slextensions.net/">SLExtensions</a></h2>  <p>Another significant Open Source project that provides a suite of controls/libraries in it’s
