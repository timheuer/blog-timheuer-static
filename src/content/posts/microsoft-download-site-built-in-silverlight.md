<p>check out the <a href="http://preview.microsoft.com/downloads">beta of the microsoft download site</a>.  it is built on <a rel="tag" href="http://silverlight.net">silverlight</a>...some media streaming happening as well as some of the subtle interactions done in <strong>silverlight</strong>.</p>
<p>if you click on the browse downloads link you can see some of it as well as other new navigation areas...</p>
<p>good to see <strong>microsoft</strong> starting to implement it in broader sites.</p>
