<p>today, the iis team <a href="http://www.iis.net/downloads/default.aspx?tabid=34&amp;g=6&amp;i=1602">released a preview of the 'web deployment tool'</a> for <strong>internet information services</strong> (iis).  the tool has features that a lot of server admins would want such as deployment assistance, sync/migration of sites across servers (which includes config, content, certs, etc.).</p>
<p>the core features are:</p>
<ul>
    <ul>
        <li>synchronization and snapshot of IIS 6/7 </li>
        <li>migration from iis 6 </li>
        <li>analysis of iis 6 installed features </li>
        <li>troubleshooting and validation features</li>
    </ul>
</ul>
<p>go <a href="http://www.iis.net/downloads/default.aspx?tabid=34&amp;g=6&amp;i=1602">check out the post</a> for all the details and a download link.</p>
