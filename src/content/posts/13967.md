<p>got a windows mobile phone?  don't want to wait until the official daylight savings time patch is available?  check out <a href="http://www.edgeblog.net/2007/daylight-saving-time-windows-mobile-fix/">edgeblog's patch</a>.  due to an energy act, daylight savings is changing in 2007.  this affects all technology vendors, including microsoft.  while a <a href="http://www.microsoft.com/windows/timezone/dst2007.mspx">patch is available for windows</a>, there is not yet one available for windows mobile, but the above patch essentially changes the registry to make it fixed. </p> <p><strong>UPDATE:</strong> added missing link</p> <p> </p><div class="wlWriterSmartContent" id="0767317B-992E-4b12-91E0-4F059A8CECA8:f1c47988-5305-45b0-bb5a-46cdb9033c08" contenteditable="false" style="padding-right: 0px; display: inline; padding-left: 0px; padding-bottom: 0px; margin: 0px; padding-top: 0px"></div>
