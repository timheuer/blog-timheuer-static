<p>The <a href="http://silverlight.live.com">Silverlight Streaming</a> service has been upgraded to support <a href="http://silverlight.net">Silverlight</a> 2 beta 2 applications.</p>  <p>As a note to customers who were previously hosting Beta 1 applications for test purposes, as beta 1 is no longer a supported test platform for SLS.  Authors should update/upload their applications using the latest Silverlight 2 bits.  These are available (with the tools) from the <a href="http://silverlight.net/GetStarted">Silverlight community site</a>. </p>  <p>Silverlight 1 applications hosted on SLS are not affected by this upgrade and still supported of course.</p>  <p>Remember that you can also now directly upload a XAP file to SLS using the <em>Manage Applications</em> functionality and then it will dynamically create a manifest file for you – so you don’t have to even worry about packaging up things in some scenarios!</p>  <div class="wlWriterSmartContent" id="scid:0767317B-992E-4b12-91E0-4F059A8CECA8:df0dd5ef-dc49-4c1e-b9a7-8ddc9c367e18" style="padding-right: 0px; display: inline; padding-left: 0px; float: none; padding-bottom: 0px; margin: 0px; padding-top: 0px"></div>
