<p>rss feedreader 2.0 (formerly rss feed web part 1.3) is now available.</p>
<p>functionality included:</p>
<ul>
<li>installer for easier installation (must be a box admin and run on the machine) 
<ul>
<li>installer installs to all portals</li></ul>
</li><li>consume multiple rss feeds and display in web part 
</li><li>implement caching (by the hours) -- default is 2 
</li><li>filter by number of past days to view 
</li><li>preview headlines (if rss feed provided)</li></ul>
<p>this is a free web part and may be used commercially, granted it remains untouched.</p>
