<p>now that silverlight is 'officially' released, maybe you've been waiting for that "RC" label to go away.  okay, no more excuses, time to start digging in.  what better way to do that then cuddle up with a few of your closest unknown friends and hack away, barcamp style!</p>
<p>some fine folks have organized a <a href="http://www.barcamp.org/SilverlightDevCampSF">SilverlightDevCamp</a> in two areas and they are free to attend and get some SWAG, some learning, and make some connections.  designers, developers, people with macbooks, everyone is welcome.</p>
<p>what is a silverlightdevcamp?  i'm glad you asked.</p>
<blockquote>
<p> </p>
<p>SilverlightDevCampSF is an upcoming gathering, inspired by <a href="http://www.barcamp.org">BarCamp</a>, to build <a href="http://silverlight.net">Silverlight</a> applications, It's organized by volunteers &amp; Silverlight enthusiasts, with attendance free to all. </p>
<p>While similar to <a href="http://www.barcamp.org/iPhoneDevCamp">iPhoneDevCamp</a> this event is focused on a singular technology, we'd like offer some diverse sessions like using PHP to generate Silverlight content, <a href="http://en.wikipedia.org/wiki/IronRuby">IronRuby</a>/<a href="http://www.codeplex.com/Wiki/View.aspx?ProjectName=IronPython">IronPython</a> &amp; the DLR or <a href="http://www.mono-project.com/Moonlight">Moonlight</a> on Linux.</p>
<p>Attendees will include web designers, developers, testers, all working together over the weekend to build Silverlight applications, share experiences, ask questions, and push the limits of web application design. If you're interested in Silverlight, but haven't had time to learn, this is your chance to get up to speed. Plus you'll be able network with other developers in the bay area who are interested in Silverlight.</p>
</blockquote>
<p>stuff will be given away and some super geeks will be on hand to help with silverlight questions.  it sounds like a great time, and in 2 great cities.  unfortunately i'm in two different cities at the same time...argh.  but there are some cool peeps that i know will be there.  if you are close, you should definitely go.  if you are working on silverlight or curious, then you should be there and make time to go.</p>
<ul>

        <li>SilverlightDevCampSF (San Francisco) - more information <a href="http://www.barcamp.org/SilverlightDevCampSF">here</a> and <a href="http://blogs.msdn.com/aniyer/archive/2007/08/27/silverlightdevcamp-on-9-14-9-15.aspx">here</a>.  <a href="http://twitter.com/SilverlightSF">follow them on twitter for updates</a>. </li>
        <li>SilverlightDevCampChicago - more information <a href="http://barcamp.org/SilverlightDevCampChicago">here</a>. </li>
    </ul>
</ul>
<p>be there.  let me know how it goes.</p>
